# 线段法


## 一、核心概念

线段法，又称“十字交叉法”，听起来可能有些抽象，但它其实源于我们生活中非常熟悉的一个场景：<span style={{color:'red'}}>“混合”</span>。比如将两种不同浓度的果汁混合在一起，得到一杯新口味的果汁。线段法就是解决这类混合问题的速算利器。

为了让你彻底理解，我们不用复杂的符号，先从一个例子入手。

**场景：调制一杯完美的果味饮料**

假设我们有两种饮料：
*   **A饮料（苹果味）**：比较酸，含糖量为 **10%**。
*   **B饮料（葡萄味）**：比较甜，含糖量为 **20%**。

我们希望将这两种饮料混合，调制出一种含糖量为 **16%** 的特调饮料。那么问题来了：<span style={{color:'orange'}}>这两种饮料应该按什么比例混合呢？</span>

在这里，我们定义两个核心要素：
- **值 (V)**：指我们关心的那个特定属性，在这个例子里就是“含糖量/浓度”。所以 $V_A = 10\%$，$V_B = 20\%$，混合后的值 $V_{mix} = 16\%$。
- **量 (M)**：指用于混合的每部分的数量，在这个例子里就是两种饮料的“重量”或“体积”。我们设需要苹果饮料的量为 $M_A$，葡萄饮料的量为 $M_B$。

### 核心原理：加权平均

所有混合问题的本质都离不开“加权平均”这个概念。总的含糖量，等于各部分含糖量之和。

$$ \text{总糖量} = \text{A饮料的糖量} + \text{B饮料的糖量} $$
$$ (M_A + M_B) * V_{mix} = M_A * V_A + M_B * V_B $$

整理一下，就得到加权平均的经典公式：

$$ V_{mix} = \frac{M_A * V_A + M_B * V_B}{M_A + M_B} $$

这个公式绝对正确，但如果用它来反推 $M_A$ 和 $M_B$ 的比例，计算起来会有点麻烦。而线段法的绝妙之处，就是对这个公式做了一个漂亮的“变形”，让我们能一眼看出比例关系。

### 2. 公式推导：从代数到“十字”的变形记

现在，我们来看看这个变形是怎么发生的。我们会用纯代数和上面的饮料例子同步进行，让你看清楚每一步的意义。

| 代数推导 | 饮料例子（代入数值） |
| :--- | :--- |
| **1. 去分母**：将公式两侧同乘以 $(M_A + M_B)$ | |
| $V_{mix} * (M_A + M_B) = M_A * V_A + M_B * V_B$ | $16\% * (M_A + M_B) = M_A * 10\% + M_B * 20\%$ |
| **2. 展开括号** | |
| $M_A * V_{mix} + M_B * V_{mix} = M_A * V_A + M_B * V_B$ | $M_A * 16\% + M_B * 16\% = M_A * 10\% + M_B * 20\%$ |
| **3. 移项**：把所有带 $M_A$ 的项移到等式一边，带 $M_B$ 的项移到另一边。 | 把含 $M_A$ 的项放左边，含 $M_B$ 的项放右边。 |
| $M_A * V_{mix} - M_A * V_A = M_B * V_B - M_B * V_{mix}$ | $M_A * 16\% - M_A * 10\% = M_B * 20\% - M_B * 16\%$ |
| **4. 提取公因式** | |
| $M_A * (V_{mix} - V_A) = M_B * (V_B - V_{mix})$ | $M_A * (16\% - 10\%) = M_B * (20\% - 16\%)$ |
| | $M_A * 6\% = M_B * 4\%$ |
| **5. 变形为比例式**：整理成比例关系！ | |
| $ \frac{M_A}{M_B} = \frac{V_B - V_{mix}}{V_{mix} - V_A} $ | $ \frac{M_A}{M_B} = \frac{20\% - 16\%}{16\% - 10\%} = \frac{4\%}{6\%} = \frac{2}{3} $ |

这个公式告诉我们一个非常重要的结论：<span style={{color: 'red'}}>两个部分的量之比，等于它们各自的值与混合值的“交叉作差”之比</span>。
在我们的例子中，$\frac{M_A}{M_B} = \frac{2}{3}$，意味着要调制出16%的饮料，苹果饮料和葡萄饮料的重量比应该是 2 : 3。

### 3. “十字交叉”可视化

为了更直观地记忆和使用这个规律，我们把它画成一个“十字交叉”的图：

```text
部分A的值 (V_A)      差值 |V_{mix} - V_B|  <-- 对应 M_A 的比例
                ╲    ╱
                  混合值 (V_{mix})
                ╱    ╲
部分B的值 (V_B)      差值 |V_A - V_{mix}|  <-- 对应 M_B 的比例
```

从这个图中，我们可以总结出那句经典口诀：<span style={{color: 'orange'}}>**部分写两边，混合写中间，交叉作差求比例，距离与量成反比。**</span>

- **距离**：指部分值与混合值的差的绝对值，即 $|V_A - V_{mix}|$ 和 $|V_{mix} - V_B|$。
- **量的反比**：A的距离 $|V_A - V_{mix}|$ 对应的是B的量 $M_B$；B的距离 $|V_{mix} - V_B|$ 对应的是A的量 $M_A$。

所以，量的比例关系为：
$$ M_A : M_B = |V_{mix} - V_B| : |V_A - V_{mix}| $$

### 4. 适用题型

只要题目符合“部分+部分=整体”的混合模型，都可以考虑使用线段法。
*   **溶液混合**：浓度为值，溶液质量为量。
*   **平均数混合**：平均分为值，人数/个数为量。
*   **增长率混合**：<span style={{color: 'red'}}>分增长率为值，基期量为量</span>（<span style={{color: 'orange'}}>注意：这里的量是基期量，不是现期量！</span>）。
*   **利润率混合**：利润率为值，成本为量。
*   **价格混合**：单价为值，数量为量。

## 二、真题讲解

### 1. 平均数混合问题
**【例1】**(2022四川下)某年级有甲、乙、丙三个班级，三个班级的期末考试平均成绩分别为70分、88分和74分。若甲班和乙班的平均成绩为78分，乙班和丙班的平均成绩为82分。问该年级的期末考试平均成绩为多少分?

- A. 75
- B. 77
- C. 79
- D. 81

<BlurredAnswer>
**【思路梳理】**

本题是典型的混合平均数问题。题目给出了三个班级中“甲乙”混合后的平均分和“乙丙”混合后的平均分，要求解的是“甲乙丙”三个班级混合后的总平均分。我们可以利用线段法，通过已知的局部混合平均分，<span style={{color: 'red'}}>两次使用线段法，反推出甲、乙、丙三个班的人数比例关系</span>，然后利用加权平均公式求最终的总平均分。

**【解题步骤】**

1.  **第一次使用线段法：求甲、乙班的人数比**
    -   部分值：甲班平均分 `70`，乙班平均分 `88`
    -   混合值：甲乙两班平均分 `78`
    -   根据“距离与量成反比”原则，量的比例等于交叉作差的比例：
        $$ \frac{\text{甲班人数}}{\text{乙班人数}} = \frac{|88 - 78|}{|78 - 70|} = \frac{10}{8} = \frac{5}{4} $$
    -   所以，甲班人数 : 乙班人数 = 5 : 4。

2.  **第二次使用线段法：求乙、丙班的人数比**
    -   部分值：乙班平均分 `88`，丙班平均分 `74`
    -   混合值：乙丙两班平均分 `82`
    -   同样，根据“距离与量成反比”原则：
        $$ \frac{\text{乙班人数}}{\text{丙班人数}} = \frac{|82 - 74|}{|88 - 82|} = \frac{8}{6} = \frac{4}{3} $$
    -   所以，乙班人数 : 丙班人数 = 4 : 3。

3.  **整合比例并赋值**
    -   从以上两步可知：甲 : 乙 = 5 : 4，乙 : 丙 = 4 : 3。
    -   乙的份数在两个比例中都是`4`，可以直接连接，得到：<span style={{color: 'red'}}>甲 : 乙 : 丙 = 5 : 4 : 3</span>。
    -   为了计算方便，我们直接赋值：甲班有5人，乙班有4人，丙班有3人。

4.  **计算总平均分**
    -   利用加权平均公式：
        $$ \text{总平均分} = \frac{\text{总分数}}{\text{总人数}} = \frac{70 * 5 + 88 * 4 + 74 * 3}{5 + 4 + 3} $$
    -   计算得：
        $$ \text{总平均分} = \frac{350 + 352 + 222}{12} = \frac{924}{12} = 77 $$

故正确答案为B。
</BlurredAnswer>

### 2. 比例混合问题
**【例2】**(2016联考)某高校艺术学院分音乐系和美术系两个系别，已知学院男生人数占总人数的30%，且音乐系男、女生人数之比为1:3，美术系男、女 生人数之比为2:3，问音乐系和美术系的总人数之比为多少?

- A. 5:2
- B. 5:1
- C. 3:1
- D. 2:1

<BlurredAnswer>
**【思路梳理】**

本题是混合比例问题，将“音乐系”和“美术系”两个部分混合得到“艺术学院”这个整体。我们可以把<span style={{color: 'red'}}>“男生占比”看作是线段法中的“值”</span>，把“各系总人数”看作是“量”。通过各部分的男生占比与混合后的总男生占比，利用线段法可以反推出音乐系和美术系的总人数之比。

**【解题步骤】**

1.  **确定各部分的值（男生占比）**
    -   音乐系男生占比：男:女=1:3，则男生占音乐系总人数的 $\frac{1}{1+3} = \frac{1}{4} = 25\%$。
    -   美术系男生占比：男:女=2:3，则男生占美术系总人数的 $\frac{2}{2+3} = \frac{2}{5} = 40\%$。
    -   混合后的值（学院总男生占比）：$30\%$。

2.  **使用线段法/十字交叉法**
    -   部分值：音乐系 `25%`，美术系 `40%`
    -   混合值：学院 `30%`
    -   根据“距离与量成反比”原则，量的比例等于交叉作差的比例：
        $$ \frac{\text{音乐系总人数}}{\text{美术系总人数}} = \frac{|40\% - 30\%|}{|30\% - 25\%|} = \frac{10\%}{5\%} = \frac{2}{1} $$

3.  **得出结论**
    -   音乐系总人数 : 美术系总人数 = 2 : 1。

故正确答案为D。
</BlurredAnswer>

### 3. 浓度混合问题
**【例3】**(2018新疆)将浓度为15%和5%的盐水各1000克，分别倒出若干配制成浓度为10%的盐水1200克，将剩下的盐水全部混合在一起，得到的盐水浓度为：

- A. 10%
- B. 8.25%
- C. 8%
- D. 7.25%

<BlurredAnswer>
**【思路梳理】**

本题为经典的溶液混合问题，分为两个过程：先是“配制”过程，后是“剩余混合”过程。
核心突破口在于：<span style={{color: 'red'}}>对于同一浓度的溶液，无论取出的量是多少，其浓度不变</span>。我们可以先分析第一个“配制”过程，用线段法求出配制1200克盐水时，两种盐水所需的质量比。然后根据初始质量，计算出剩余的两种盐水质量，最后再对剩余部分进行一次混合计算。

**【解题步骤】**

1.  **分析“配制”过程，求所用盐水的质量比**
    -   部分值（浓度）：A盐水 `15%`，B盐水 `5%`
    -   混合值（浓度）：`10%`
    -   使用线段法求配制时两种盐水的质量之比：
        $$ \frac{\text{15\%盐水的质量}}{\text{5\%盐水的质量}} = \frac{|10\% - 5\%|}{|15\% - 10\%|} = \frac{5\%}{5\%} = \frac{1}{1} $$
    -   这说明，要配制出10%的盐水，需要等质量的15%盐水和5%盐水混合。

2.  **计算配制时各自用掉的质量**
    -   总共配制了1200克盐水，且两种盐水质量比为1:1。
    -   所以，用掉了15%的盐水 $1200 * \frac{1}{2} = 600$ 克。
    -   用掉了5%的盐水 $1200 * \frac{1}{2} = 600$ 克。

3.  **计算剩余盐水的质量**
    -   两种盐水初始各有1000克。
    -   剩余的15%盐水质量：$1000 - 600 = 400$ 克。
    -   剩余的5%盐水质量：$1000 - 600 = 400$ 克。

4.  **计算“剩余混合”后的浓度**
    -   现在我们把剩下的400克15%的盐水和400克5%的盐水混合。
    -   由于<span style={{color: 'orange'}}>剩余的两种盐水质量依然相等 (400克 = 400克)</span>，它们的混合浓度等于浓度的算术平均值。
        $$ \text{最终浓度} = \frac{\text{总溶质}}{\text{总溶液}} = \frac{400 * 15\% + 400 * 5\%}{400 + 400} = \frac{400 * (15\% + 5\%)}{800} = \frac{20\%}{2} = 10\% $$

故正确答案为A。

**【秒杀技巧】**
由于配制10%的盐水需要1:1质量的15%和5%的盐水，而已知初始时两种盐水质量也是1:1（各1000克）。这意味着无论倒出多少来配制，只要配制成了，倒出的两种盐水质量必然是1:1。那么，剩下的两种盐水质量也必然是1:1。既然质量比不变，再次混合后的浓度自然也不会变，仍然是10%。
</BlurredAnswer>

### 4. 增长率混合问题
**【例4】**(2023上海·A类)某公司生产A、B两种产品，其中B是A的升级产品。经过调研，预判2022年市场对A产品的需求比2021年下降30%(A产品的价格不变)。因此公司决定增加对B产品营销，使B产品在2022年的销售收入比2021年增长70%，这样恰好使公司2022年的总销售收入比2021年增长10%。则2021年B产品的销售额占总销售额的比例是：

- A. 40%
- B. 50%
- C. 60%
- D. 70%

<BlurredAnswer>
**【思路梳理】**

本题是典型的增长率混合问题。公司总销售收入的增长率，是A、B两种产品销售收入增长率的加权平均。我们可以把<span style={{color: 'red'}}>各产品的“增长率”看作是“值”</span>，把它们在<span style={{color: 'red'}}>“基期（2021年）的销售额”看作是“量”</span>。通过线段法，可以求出2021年A、B两产品销售额的比，进而求得B产品销售额的占比。

**【解题步骤】**

1.  **确定各部分的值（增长率）**
    -   A产品：价格不变，需求下降30%，所以销售收入增长率为 `-30%`。
    -   B产品：销售收入增长率为 `+70%`。
    -   混合后（总收入）的增长率：`+10%`。

2.  **使用线段法/十字交叉法**
    -   部分值：A产品 `-30%`，B产品 `+70%`
    -   混合值：总收入 `+10%`
    -   根据“距离与量成反比”原则，基期量（2021年销售额）的比例等于交叉作差的比例：
        $$ \frac{\text{A产品2021年销售额}}{\text{B产品2021年销售额}} = \frac{|70\% - 10\%|}{|10\% - (-30\%)|} = \frac{60\%}{40\%} = \frac{3}{2} $$
    -   <span style={{color: 'orange'}}>注意</span>：在计算与负数的距离时，`10% - (-30%)` 等于 `10% + 30%`。

3.  **计算B产品销售额占比**
    -   我们得到2021年A、B两产品销售额之比为 3 : 2。
    -   总销售额的份数就是 $3 + 2 = 5$ 份。
    -   B产品销售额占总销售额的比例为：
        $$ \frac{\text{B产品销售额}}{\text{总销售额}} = \frac{2}{3 + 2} = \frac{2}{5} = 40\% $$

故正确答案为A。
</BlurredAnswer>

### 5. 混合增长率问题
**【例5】**(2021江苏)2019年，全国居民人均消费支出21559元，比2012年增长78.9%。其中，城镇居民人均消费支出28063元，比2012年增长64.0%；农村居民人均消费支出13328元，比2012年增长99.9%。则2019年城镇居民人口占总人口的比重约为：

- A. 52.7%
- B. 53.8%
- C. 54.1%
- D. 55.9%

<BlurredAnswer>
**【思路梳理】**

本题是资料分析中的混合平均数问题。全国居民的人均消费支出，是城镇和农村居民人均消费支出的加权平均值。我们可以把<span style={{color: 'red'}}>“人均消费支出”看作是“值”</span>，把<span style={{color: 'red'}}>“人口数”看作是“量”</span>。通过线段法，可以反推出城镇和农村的人口之比，进而计算城镇人口占比。题干中给出的增长率信息是无关的干扰项。

**【解题步骤】**

1.  **确定各部分的值（人均消费支出）**
    -   城镇居民（部分1）：`28063` 元
    -   农村居民（部分2）：`13328` 元
    -   全国居民（混合后）：`21559` 元

2.  **使用线段法/十字交叉法**
    -   根据“距离与量成反比”原则，人口数的比例等于交叉作差的比例：
        $$ \frac{\text{城镇人口数}}{\text{农村人口数}} = \frac{|21559 - 13328|}{|28063 - 21559|} = \frac{8231}{6504} $$

3.  **估算比例并计算占比**
    -   直接计算 `8231 / 6504` 比较复杂，我们可以进行估算。`8231 ≈ 8200`，`6504 ≈ 6500`。
    -   比例约为 $8200 : 6500 = 82 : 65$。
    -   城镇人口占总人口的比重为：
        $$ \frac{\text{城镇人口数}}{\text{总人口数}} = \frac{\text{城镇人口数}}{\text{城镇人口数} + \text{农村人口数}} \approx \frac{82}{82 + 65} = \frac{82}{147} $$
    -   进行除法估算：$\frac{82}{147} \approx \frac{82}{150} = \frac{164}{300} \approx 54.7\%$。或者 $\frac{82}{147}$ 比 $\frac{82}{140} = \frac{41}{70} \approx 58\%$ 略小，比 $\frac{82.5}{150} = \frac{165}{300} = 55\%$ 略大。最接近的选项是55.9%。
    -   精确计算：$82 \div 147 \approx 0.5578$，即 `55.8%`。

故正确答案为D。
</BlurredAnswer>

### 6. 基期现期混合问题
**【例6】**(2020山东)由于改良了种植技术，农场2017年种植的A和B两种作物，产量分别增加了10%和25%。已知2017年两种作物总产量增加了18%，则2017年A和B两种作物的产量比为：

- A. 7:8
- B. 8:7
- C. 176:175
- D. 77:100

<BlurredAnswer>
**【思路梳理】**
本题是混合增长率问题，但有一个<span style={{color: 'orange'}}>关键陷阱</span>：线段法求出的是**基期量**之比，而题目问的是**现期量**之比。
我们的步骤是：
1.  用线段法，通过增长率（值）求出基期（2016年）产量（量）的比。
2.  根据基期产量比和各自的增长率，计算出现期（2017年）的产量比。

**【解题步骤】**

1.  **第一步：用线段法求基期（2016年）产量比**
    -   部分值（增长率）：A作物 `10%`，B作物 `25%`
    -   混合值（总增长率）：`18%`
    -   根据“距离与量成反比”原则，基期产量的比等于交叉作差的比：
        $$ \frac{\text{A作物2016年产量}}{\text{B作物2016年产量}} = \frac{|25\% - 18\%|}{|18\% - 10\%|} = \frac{7\%}{8\%} = \frac{7}{8} $$
    -   所以，2016年A、B的产量比为 7 : 8。

2.  **第二步：根据基期比计算现期（2017年）产量比**
    -   为了计算方便，我们赋值：2016年A产量为`7`份，B产量为`8`份。
    -   计算2017年的产量：
        -   2017年A产量 = $7 * (1 + 10\%) = 7 * 1.1 = 7.7$ 份
        -   2017年B产量 = $8 * (1 + 25\%) = 8 * 1.25 = 10$ 份
    -   计算2017年A、B的产量比：
        $$ \frac{\text{A作物2017年产量}}{\text{B作物2017年产量}} = \frac{7.7}{10} = \frac{77}{100} $$
    -   所以，2017年A和B两种作物的产量比为 77 : 100。

故正确答案为D。

## 三、技巧总结

线段法（十字交叉法）是解决各类“混合问题”的利器，其核心在于将复杂的代数运算转化为直观的比例关系。掌握其精髓，能大幅提升解题速度。

### 1. 核心思想

<span style={{color:'red'}}>**抓住“混合”本质，万物皆可“交叉”**</span>。
所有“部分+部分=整体”且涉及加权平均计算的题型，理论上都可以尝试使用线段法。你需要做的就是准确找到问题中的三个核心值：**部分值A**、**部分值B** 和 **混合值**。

### 2. 标准使用步骤

1.  **识别模型**：判断题目是否属于混合问题，即两个部分（或多个部分）融合成一个整体。
2.  **找准“值”与“量”**：
    -   **“值”**：指那个可以被平均的属性，如<span style={{color:'orange'}}>浓度、平均分、增长率、利润率、价格</span>等。
    -   **“量”**：指与“值”相对应的权重，如<span style={{color:'orange'}}>溶液质量、人数、基期量、成本、数量</span>等。
3.  **画图交叉**：将两个部分值写在十字的左侧上下两端，混合值写在中间。
4.  **交叉作差**：用两边的部分值分别与中间的混合值作差，将差的绝对值写在十字的右侧。
5.  **得出比例**：右侧得到的差值之比，就是左侧两个部分对应“量”的<span style={{color:'red'}}>反比</span>。

### 3. 关键要点与高频易错点

1.  **增长率问题必考陷阱**
    -   使用线段法处理增长率问题时，交叉作差得到的是<span style={{color:'red'}}>**基期量之比**</span>，而非现期量之比！
    -   如果题目问的是**现期量**之比（如例6），必须在求出基期量之比后，再乘以各自的增长系数 `(1+r)`，才能得到最终答案。

2.  **“量”的准确对应**
    -   **利润率** 对应的“量”是 **成本**，不是售价。
    -   **增长率** 对应的“量”是 **基期量**，不是现期量。
    -   **单价** 对应的“量”是 **数量**。
    -   这个对应关系一旦出错，全盘皆输。

3.  **作差的细节**
    -   当遇到负增长率时（如例4的-30%），作差要格外小心。`|10% - (-30%)|` 变成 `10% + 30%`。
    -   作差永远是**大数减小数**，保证结果为正。

4.  **信息干扰**
    -   题目中可能给出多余信息（如例5中的年份和增长率），旨在迷惑考生。要能迅速识别出进行交叉计算所需的核心值，排除干扰。

### 4. 秒杀技巧与特殊应用

1.  **中点秒杀**
    -   如果混合后的“值”恰好是两个部分“值”的**算术平均中点**，那么说明两个部分的“量”**必然相等**，比例为 1:1。反之亦然。

2.  **比例思想**
    -   在溶液问题中（如例3），如果初始时两种溶液质量比为 `a:b`，配制出的溶液也要求两种溶液质量比为 `a:b`，那么<span style={{color:'orange'}}>无论倒出多少，剩余两种溶液的质量比仍然是 a:b</span>，再次混合后浓度不变。

3.  **多量混合**
    -   当遇到三个或更多个量混合时，可以采用“假设法”或“两两结合”的方式。先将其中两个混合，得到一个中间值，再用这个中间值与第三个量进行混合。
