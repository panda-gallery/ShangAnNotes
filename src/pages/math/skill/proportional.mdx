## 一、核心概念：比例法

### 1. 什么是比例法？

比例法是一种通过分析题目中各个量之间存在的比例关系，来简化计算、快速求解的数学方法。在公务员考试的“数量关系”模块，很多题目看似复杂，但其内在都遵循着一定的比例关系。掌握了比例法，就如同拥有了一把解开复杂问题的钥匙。

**举个身边的例子：**

假设你去商店买苹果，每个苹果5元钱。

- 你买1个，总价是5元。
- 你买2个，总价是10元。
- 你买10个，总价是50元。

在这个过程中，苹果的 **“单价”** 是一个固定不变的量。而 **“总价”** 和你购买的 **“数量”** 是两个变化的量。但它们之间存在一种固定的关系：
$$ \frac{总价}{数量} = 单价(5元) $$
总价和数量的比值是一个定值（也就是单价），我们就说“总价”和“数量”成 **正比例** 关系。

现在换一种情况，假设学校食堂有一项固定的清洁任务，需要80个“工时”才能完成（比如，1个人干80小时）。

- 如果派1个学生去，需要80小时。
- 如果派2个学生去，只需要40小时。
- 如果派8个学生去，只需要10小时。

这里的 **“总工作量”** 是固定不变的。而 **“工作人数”** 和 **“工作时间”** 是变化的量。它们的关系是：
$$ 工作人数 \times 工作时间 = 总工作量(80工时) $$
工作人数和工作时间的乘积是一个定值，我们就说“工作人数”和“工作时间”成 **反比例** 关系。

### 2. 比例法的核心：寻找不变的量

<span style={{color:'red'}}>比例法的灵魂在于“寻找不变量”</span>。在题目描述的场景变化中，总有一些量是恒定不变的，或者总量、或者差值、或者某个单一的量。一旦找到了这个“不变量”，我们就可以围绕它建立比例关系，从而看清其他变量是如何变化的。

- **公式形态一（正比例）：** $A \div B = k$ (k为定值)
  - 当k不变时，A和B成正比例。A扩大N倍，B也扩大N倍。

- **公式形态二（反比例）：** $A \times B = k$ (k为定值)
  - 当k不变时，A和B成反比例。A扩大N倍，B则缩小为原来的 $\frac{1}{N}$。

**常见的三量关系：**
- 路程 = 速度 × 时间
- 总价 = 单价 × 数量
- 工作总量 = 效率 × 时间
- 溶液总质量 = 溶质质量 ÷ 浓度
- 利润 = 成本 × 利润率

在这些公式中，只要有一个量是固定的，另外两个量就会呈现出明确的正比例或反比例关系。

### 3. 比例法常用技巧

#### 3.1 统一比例：三量比例

当遇到甲:乙、乙:丙这种形式时，需要通过中间量“乙”作为桥梁，来统一甲、乙、丙三者的比例。

**推导过程：**
若 甲:乙 = a:b，乙:丙 = c:d
为了统一乙的份数，我们将第一个比例两边同时乘以c，第二个比例两边同时乘以b。
- 甲:乙 = ac : bc
- 乙:丙 = bc : bd
于是得到：**甲 : 乙 : 丙 = ac : bc : bd**

**例：** 甲:乙=2:3，乙:丙=4:5。求甲:乙:丙。
- 乙在两个比例中分别是3和4，3和4的最小公倍数是12。
- 第一个比例扩大4倍：(2×4):(3×4) = 8:12
- 第二个比例扩大3倍：(4×3):(5×3) = 12:15
- 所以，甲:乙:丙 = 8:12:15

#### 3.2 差值比例与等比性质

由基础比例式 $a:b = c:d$ (即 $\frac{a}{b} = \frac{c}{d}$ )，可以推导出更丰富的比例关系。

一个常用的变形是 **合比与分比**，它表明：
$$ \frac{a+b}{b} = \frac{c+d}{d} $$
以及
$$ \frac{a-b}{b} = \frac{c-d}{d} $$

另一个强大的工具是 **等比性质**：
$$ \frac{a}{b} = \frac{c}{d} = \frac{a+c}{b+d} = \frac{a-c}{b-d} $$

这个性质在处理“价格差”、“年龄差”等问题时非常有用。

#### 3.3 恒值问题

恒值问题是比例法的核心应用，关键是<span style={{color:'orange'}}>识别题目中哪个量在整个变化过程中是保持不变的</span>，并以此为基准进行计算。

- **和不变**：如两种溶液混合，总溶液的质量不变。
- **差不变**：如两人之间的年龄差，或者两件商品降价或涨价相同的金额，其价格差不变。
- **单个量不变**：如溶液中加水，溶质的量不变；或者甲给乙一部分钱，但甲乙的总钱数不变。

## 二、真题讲解

### 1. 多对象关联问题

**例1**
某企业为全体员工定制工作服，请服装公司的裁缝量体裁衣。裁缝每小时为52名男员工和35名女员工量尺寸。几小时后，刚好量完所有女员工的尺寸，这时还有24名男员工没量。若男员工与女员工的人数比为11：7，则该企业共有（ ）名员工。
- A.720
- B.810
- C.900
- D.1080

<BlurredAnswer>
**解法一：比例统一与差值法**

<span style={{color:'red'}}>第一步：统一参照量。</span>
题目涉及“人数”和“量衣速度”两个比例。为了方便比较，我们需要将这两个比例统一。
- 已知男女人数比为 `11 : 7`。
- 已知男女量衣速度比为 `52 : 35`。
注意到“刚好量完所有女员工”，我们可以以女员工为基准。女员工人数是7份，量衣速度是35人/小时，我们可以将人数比扩大5倍，使其与速度产生关联。
- 统一后的人数比：`11×5 : 7×5` => <span style={{color:'blue'}}>`55 : 35`</span>。
- 速度比为：<span style={{color:'blue'}}>`52 : 35`</span>。

<span style={{color:'red'}}>第二步：计算比例差值。</span>
现在，我们可以这样理解：假设在一个单位时间内，裁缝的目标是量完35份女员工和55份男员工。但他们的实际速度只能量完35份女员工和52份男员工。
- 这意味着，在量完所有女员工的相同时间内，男员工的测量进度会“落后”。
- 每完成35份女员工的测量，男员工就差 `55 - 52 = 3` 份没有测量。

<span style={{color:'red'}}>第三步：计算总时长和总人数。</span>
题目中，最终剩下24名男员工没量，这24人就对应着落后的 `3` 份。
- 每一份对应的人数是：`24 ÷ 3 = 8` 人。
- 企业总人数对应的总份数是 `55 + 35 = 90` 份。
- 所以企业总员工数是：`90 × 8 = 720` 人。

**解法二：倍数特性法（秒杀技巧）**

<span style={{color:'red'}}>第一步：利用比例关系锁定总人数范围。</span>
男员工与女员工的人数比为11:7，说明总人数一定是 `11 + 7 = 18` 的倍数。
- 检验选项：A(720/18=40), B(810/18=45), C(900/18=50), D(1080/18=60)。所有选项都是18的倍数，此路不通，需结合其他条件。

<span style={{color:'red'}}>第二步：利用余数关系进行排除。</span>
“量完所有女员工时，还剩24名男员工没量”，这句话是关键。
- 这说明，总员工人数减去24，剩下的员工数就是被裁缝量完的。
- 量完的员工总数 = `总人数 - 24`。
- 裁缝是按 `52+35=87` 人的组合来工作的，所以量完的员工数一定是`87`的倍数。
- 我们来检验选项：
  - A. `720 - 24 = 696`。 `696 ÷ 87 = 8`。符合。
  - B. `810 - 24 = 786`。 `786 ÷ 87` 不是整数。排除。
  - C. `900 - 24 = 876`。 `876 ÷ 87` 不是整数。排除。
  - D. `1080 - 24 = 1056`。`1056 ÷ 87` 不是整数。排除。
- 因此，只有A选项满足所有条件。

</BlurredAnswer>

### 2. 恒值问题

#### 2.1 和不变：溶液混合

**例2**
一个瓶中酒精与水的体积比是3：1，另一个瓶中酒精与水的体积比是4：1，若把两瓶酒精溶液混合，则混合后的酒精和水的体积之比是多少？
- A.31：9
- B.7：2
- C.31：40
- D.20：11

<BlurredAnswer>
**解法：抓住“溶液总量”这个不变量进行统一**

<span style={{color:'red'}}>第一步：分析原始比例和溶液总量。</span>
- 第一瓶：酒精:水 = 3:1，总溶液份数为 `3+1=4` 份。
- 第二瓶：酒精:水 = 4:1，总溶液份数为 `4+1=5` 份。

<span style={{color:'red'}}>第二步：统一溶液总量的份数。</span>
为了能够直接相加，我们需要让两瓶溶液的“总份数”一致。我们取4和5的最小公倍数20，作为统一后的标准份数。
- **第一瓶**：总量从4份变为20份（扩大5倍），则酒精和水也扩大5倍。
  - 酒精:水 = `(3×5) : (1×5)` = `15 : 5`。
- **第二瓶**：总量从5份变为20份（扩大4倍），则酒精和水也扩大4倍。
  - 酒精:水 = `(4×4) : (1×4)` = `16 : 4`。

<span style={{color:'red'}}>第三步：合并计算最终比例。</span>
现在两瓶溶液的总量都是20份，可以相加了。
- 混合后总酒精 = `15 + 16 = 31` 份。
- 混合后总水量 = `5 + 4 = 9` 份。
- 最终比例为 `31 : 9`。

</BlurredAnswer>

### 2.2 差不变：价格升降

**例3**
甲乙两种商品的价格比是3：5，如果他们的价格分别下降50元，它们的价格比是4：7，这两种商品原来的价格各为多少元？
- A.300元 500元
- B.375元 625元
- C.450元 750元
- D.525元 875元

<BlurredAnswer>
**解法一：恒值比例法，抓住“价格差”不变**

<span style={{color:'red'}}>第一步：分析不同时期的比例和差值。</span>
- 降价前：价格比是 3:5，价格差是 `5-3=2` 份。
- 降价后：价格比是 4:7，价格差是 `7-4=3` 份。
- 核心不变量：因为两件商品都下降了50元，所以它们之间的<span style={{color:'orange'}}>价格差</span>始终是不变的。

<span style={{color:'red'}}>第二步：统一“价格差”的份数。</span>
为了让价格差在数值上相等，我们取2和3的最小公倍数6，作为统一后的差值份数。
- **降价前**：差值从2份变为6份（扩大3倍），则原价格比也扩大3倍。
  - 价格比 = `(3×3) : (5×3)` = `9 : 15`。
- **降价后**：差值从3份变为6份（扩大2倍），则后价格比也扩大2倍。
  - 价格比 = `(4×2) : (7×2)` = `8 : 14`。

<span style={{color:'red'}}>第三步：计算每份的实际价值。</span>
现在我们来看甲商品，它的价格从9份变成了8份，减少了 `9-8=1` 份。
- 这一份的减少量，正是题目中说的下降了50元。所以 `1份 = 50元`。
- 原来甲的价格是9份，即 `9 × 50 = 450` 元。
- 原来乙的价格是15份，即 `15 × 50 = 750` 元。

**解法二：数字特性法（快速验证）**

- 两种商品价格都减去50后，新的价格比为4:7。
- 检验选项C：450-50=400，750-50=700。 `400 : 700` 化简后为 `4:7`。符合题意。

</BlurredAnswer>

#### 2.3 单一量不变：加水稀释

**例4**
千禧锻造厂要制造一批一定比例的锡铁金属合金，第一次加入适量的金属铁后，此时金属锡的含量占总重量的4%，第二次加入同样多的金属铁后，金属锡的含量占总重3%，如果第三次再加入同样多的金属铁后，此时金属锡的含量占总重量的百分比是（ ）
- A.2.5%
- B.2.4%
- C.2.7%
- D.2.8%

<BlurredAnswer>
**解法：恒值比例法，抓住“锡的量”始终不变**

<span style={{color:'red'}}>第一步：分析比例关系，找到不变量。</span>
- 整个过程中，只加入了铁，<span style={{color:'orange'}}>金属锡的质量</span> 是一个恒定不变的量。
- 第一次：锡占4%，即 $\frac{4}{100}$，锡 : 总重 = 4 : 100 = 1 : 25。
- 第二次：锡占3%，即 $\frac{3}{100}$，锡 : 总重 = 3 : 100。

<span style={{color:'red'}}>第二步：统一“锡”的份数。</span>
为了比较总重量的变化，我们需要将锡的份数统一。取1和3的最小公倍数3。
- **第一次**：锡的份数从1变为3（扩大3倍），则总重也扩大3倍。
  - 锡 : 总重 = `(1×3) : (25×3)` = `3 : 75`。
- **第二次**：锡的份数是3，保持不变。
  - 锡 : 总重 = `3 : 100`。

<span style={{color:'red'}}>第三步：分析变化量，推导第三次的结果。</span>
- 从第一次到第二次，锡的份数都是3，总重量从75份增加到100份，增加了 `100 - 75 = 25` 份。
- 这25份就是第二次加入的“同样多的金属铁”。
- 第三次再加入同样多的铁，总重量会再增加25份，变为 `100 + 25 = 125` 份。
- 此时，锡的量仍然是3份，总重量是125份。
- 第三次的锡含量为：$\frac{3}{125} = \frac{24}{1000} = 2.4\%$。

</BlurredAnswer>

#### 2.4 和不变但内部分配变化

**例5**
股民甲和乙分别持有同一家公司的股票。如果乙将自己的10000股转给甲，则此时甲持有该股票的份额是乙的3倍；如果甲将自己的1000股转给乙，则此时乙持有该股票的份额比甲多6倍。那么，甲乙二人共持有( )股该公司股票。
- A.6400
- C.17800
- B.17600
- D.28800

<BlurredAnswer>
**解法一：恒值比例法，抓住“股票总和”不变**

<span style={{color:'red'}}>第一步：分析两种情况下的比例和总份数。</span>
- 不管是甲转给乙，还是乙转给甲，两人的<span style={{color:'orange'}}>股票总和</span>是恒定不变的。
- **情况一**：乙转给甲，甲:乙 = 3:1。总份数为 `3+1=4` 份。
- **情况二**：甲转给乙，乙比甲多6倍，即乙是甲的7倍，甲:乙 = 1:7。总份数为 `1+7=8` 份。

<span style={{color:'red'}}>第二步：统一“股票总和”的份数。</span>
取4和8的最小公倍数8，作为统一后的总份数。
- **情况一**：总份数从4变为8（扩大2倍），则甲乙比例也扩大2倍。
  - 甲:乙 = `(3×2) : (1×2)` = `6 : 2`。
- **情况二**：总份数是8，保持不变。
  - 甲:乙 = `1 : 7`。

<span style={{color:'red'}}>第三步：分析甲的份数变化，求解。</span>
- 对比两种情况，甲的持股从6份变为1份，减少了 `6-1=5` 份。
- 这种变化是怎么发生的？从情况一（乙给甲10000股）到情况二（甲给乙1000股），甲的持股量实际减少了 `10000 + 1000 = 11000` 股。
- 这11000股就对应着5份的变化量。
- 每一份对应的股票数是：`11000 ÷ 5 = 2200` 股。
- 股票总和是8份，所以总股数是 `8 × 2200 = 17600` 股。

**解法二：方程法**

<span style={{color:'red'}}>第一步：设定未知数。</span>
假设甲的真实持股为J，乙的真实持股为Y。

<span style={{color:'red'}}>第二步：根据两种情况列出方程组。</span>
- 情况一：`J + 10000 = 3 × (Y - 10000)`
- 情况二：`Y + 1000 = 7 × (J - 1000)` (乙比甲多6倍，即乙是甲的7倍)

<span style={{color:'red'}}>第三步：解方程组。</span>
- 由一式得：`J - 3Y = -40000`
- 由二式得：`7J - Y = 8000`
- 将二式乘以3：`21J - 3Y = 24000`
- 用此式减去一式：`20J = 64000` => `J = 3200`
- 代入`7J - Y = 8000`，得`7*3200 - Y = 8000` => `22400 - Y = 8000` => `Y=14400`
- 总股数 = `J + Y = 3200 + 14400 = 17600`。

</BlurredAnswer>

### 3. 基础比例应用

#### 3.1 效率与时间的反比关系

**例6**
甲乙两辆清洁车执行东西两城的公路清洁任务，甲车单独清扫需10小时，乙车单独清扫需15小时，两车同时从东西两城相向开出，相遇时甲车比乙车多清扫15千米，东西两城相距多少千米?
- A.60
- B.75
- C.90
- D.135

<BlurredAnswer>
**解法：时间与效率的反比关系**

<span style={{color:'red'}}>第一步：由时间比求效率比。</span>
- 甲乙单独完成工作的时间比为 `10 : 15`，化简得 `2 : 3`。
- 工作总量固定时，效率和时间成反比。所以，甲乙的效率比为 `3 : 2`。

<span style={{color:'red'}}>第二步：分析路程分配。</span>
- 两车同时出发，相遇时所用时间相同。
- 当时间相同时，路程与效率成正比。所以，相遇时甲乙两车清扫的路程比也为 `3 : 2`。

<span style={{color:'red'}}>第三步：利用路程差求解。</span>
- 甲比乙多清扫的路程，在比例上对应 `3 - 2 = 1` 份。
- 题目中，这1份的差值是15千米。所以 `1份 = 15千米`。
- 东西两城的总距离是甲乙路程之和，对应 `3 + 2 = 5` 份。
- 总距离为 `5 × 15 = 75` 千米。

</BlurredAnswer>

#### 3.2 速度与时间的连锁比例

**例7**
一人从甲地到乙地，步行速度比骑车速度慢75%，骑车速度比公交慢50%，如果一个人坐公车从甲地到乙地，再从乙地步行回到甲地一共用了一个半小时，则该人骑车从甲地到乙地需要多长时间?

<BlurredAnswer>
**解法：赋值法与比例法结合**

<span style={{color:'red'}}>第一步：通过比例关系给速度赋值。</span>
- “步行比骑车慢75%”，即步行速度是骑车速度的 `1 - 75% = 25% = 1/4`。
- “骑车比公交慢50%”，即骑车速度是公交速度的 `1 - 50% = 50% = 1/2`。
- 我们可以赋值：设公交速度为`8`，则骑车速度为`4`，步行速度为`1`。
- 三种方式的速度比为：步行:骑车:公交 = `1 : 4 : 8`。

<span style={{color:'red'}}>第二步：利用时间关系求解。</span>
- 从甲到乙坐公交，从乙到甲步行，路程相同。
- 当路程相同时，时间与速度成反比。
- 公交与步行的速度比是 `8 : 1`，所以花费的时间比是 `1 : 8`。

<span style={{color:'red'}}>第三步：计算实际时间。</span>
- 总时间为 `1.5小时 = 90分钟`。这90分钟被分成了 `1+8=9` 份。
- 每一份的时间是 `90 ÷ 9 = 10` 分钟。
- 所以，坐公交去乙地用时 `1份 = 10分钟`，步行回甲地用时 `8份 = 80分钟`。

<span style={{color:'red'}}>第四步：推算骑车时间。</span>
- 已知公交用时10分钟，公交与骑车的速度比是 `8 : 4 = 2 : 1`。
- 路程相同，时间与速度成反比，所以时间比为 `1 : 2`。

## 三、技巧总结

1.  **先看比例，再看问题**：读题时，优先识别题目中出现的“几比几”、“是…的几倍”、“百分之几”等词语，这些都是比例法的信号。
2.  **找准不变量是核心**：解题前先问自己，在这个过程中，什么是不变的？是总和、差值、还是某个特定物体的量？抓住不变量，就抓住了破题的关键。
3.  **份数思想代替具体数值**：不要过早代入具体数字计算，用“份数”来思考。将比例关系看作是份数的分配，可以大大简化思路。
4.  **统一比例是前提**：当出现多个比例关系时（如三量比例），必须先通过中间量将所有比例统一到同一个标准下，才能进行后续的加减比较。
5.  **灵活运用正反比**：
    - **“一同”则正**：$A=B \times C$，若A固定，B和C成反比；若B固定，A和C成正比。可以记作“一固（定）一正（比）”。
    - **“相遇/追及”**：时间相同，路程和速度成正比。
    - **“往返/环绕”**：路程相同，时间和速度成反比。
6.  **方程法是备用选项**：当比例关系非常复杂，或者找不到不变量时，方程法是最稳妥的最后防线。但通常，比例法会更快。
