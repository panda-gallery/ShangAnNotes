# 最值问题

## 一、核心概念

### 1. 最值问题分类与解题方法

最值问题分为三类：最不利构造、数列构造、多集合反向构造。<span style={{color:'red'}}>核心解题思路为极端分析法</span>（构造符合条件的极端数值）

#### (1) 最不利构造问题

**题型特征**：题干含<span style={{color:'red'}}>"至少…保证…"</span>

**解题方法**：
- 最不利情况数 = 所有不满足目标的情况总数
- 答案 = 最不利情况数 + 1

**核心思想**：<span style={{color:'red'}}>考虑所有可能的不利情况，再加1个确保目标达成</span>

#### (2) 数列构造问题

**题型特征**：求<span style={{color:'red'}}>"某个主体排名第几时最值"</span>（如"第三名最少得几分"）或<span style={{color:'red'}}>"最……最……"</span>

**解题步骤**：
1. **排序定位**：设所求量为x
2. **反向构造**：
   - 求最大值→其他量尽可能小
   - 求最小值→其他量尽可能大
3. **加和求解**：根据总和列方程
4. **取整规则**：<span style={{color:'red'}}>求最小值时向上取整；求最大值时向下取整</span>

#### (3) 多集合反向构造问题

**题型特征**：求<span style={{color:'red'}}>"全部满足的至少有多少"</span>（如"四科目都喜欢的最少人数"）

**解题方法**：利用容斥原理的反向思维，先求每项不满足的数量

### 2. 解题核心思想演示

我们通过一个生活中的例子来理解数列构造问题的核心思想：

**场景模拟：分糖果**

假设有20颗糖果，要分给小明、小红、小刚三个小朋友，要求每个小朋友至少分到1颗，且三个人分到的糖果数量各不相同。问：分到糖果最多的小朋友，**最多**能分到多少颗？**最少**能分到多少颗？

#### 求"最多"的最大值（最多的小朋友最多能分到几颗）

- **核心思想**：<span style={{color:'red'}}>要让一个量最大，就让其他量尽可能小</span>
- **构造过程**：想让"最多的小朋友"分到的糖果数量达到最大，就需要让另外两个小朋友分到的糖果数量尽可能少
- **具体分析**：
    - 根据约束条件，每人至少1颗，且数量不同
    - 让小红拿1颗，小刚拿2颗（因为数量要不同）
    - 此时，剩下的糖果都给小明：$$20 - 1 - 2 = 17$$ 颗
- **结论**：最多的小朋友<span style={{color:'red'}}>最多</span>可以分到17颗

#### 求"最多"的最小值（最多的小朋友最少能分到几颗）

- **核心思想**：<span style={{color:'red'}}>要让几个量的最大值变小，就要让这几个量尽可能地接近（平均）</span>
- **构造过程**：
    - 总共20颗糖，分给3个人，平均每人约 $$20 \div 3 ≈ 6.67$$ 颗
    - 为了让数量尽量接近且各不相同，设三人分到 $$x, y, z$$ 颗，且 $$x > y > z ≥ 1$$
    - 列方程：$$x + y + z = 20$$
    - 因为 $$x > y > z$$，所以 $$y ≤ x-1, z ≤ y-1 ≤ x-2$$
    - 代入不等式：$$x + (x-1) + (x-2) ≥ 20$$
    - 解得：$$3x ≥ 23 \implies x ≥ 7.67$$
    - <span style={{color:'red'}}>取整规则</span>：因为求最小值，所以向上取整，$$x = 8$$
    - 验证：当$$x=8$$时，$$y+z=12$$，可取$$y=7, z=5$$，满足条件
- **结论**：最多的小朋友<span style={{color:'red'}}>最少</span>可以分到8颗

## 二、真题讲解

### 数列构造

一项测验共有29道单项选择题，答对得5分，答错减3分，不答不得分也不减分，答对15题及以上另加10分，否则另减5分。小郑答题共得60分，问他最少有几道题未答？

- A. 1                
- B. 2 
- C. 3                
- D. 4

<BlurredAnswer>
**解题步骤**：

**1. 排序定位**：设答对 $$x$$ 道，答错 $$y$$ 道，未答 $$z$$ 道，题目求 $$z$$ 的最小值

**2. 反向构造**：求 $$z$$ 最小值，根据 $$z = 29 - (x+y)$$，需要让 $$x+y$$（答题总数）尽可能大

**3. 加和求解**：根据得分分情况列方程
- 情况一：$$x ≥ 15$$，得分方程：$$5x - 3y + 10 = 60 \implies 5x - 3y = 50$$
- 情况二：$$x < 15$$，得分方程：$$5x - 3y - 5 = 60 \implies 5x - 3y = 65$$

对于情况一：$$5x - 3y = 50$$
- 因为 $$5x = 50 + 3y$$，所以 $$y$$ 必须是5的倍数
- 尝试 $$y = 10$$：$$x = 16$$，满足 $$x ≥ 15$$，此时 $$z = 29 - 16 - 10 = 3$$

对于情况二：$$5x - 3y = 65$$
- 尝试 $$y = 0$$：$$x = 13$$，满足 $$x < 15$$，此时 $$z = 29 - 13 - 0 = 16$$

**4. 取整规则**：求最小值，比较两种情况得 $$z = 3$$

**答案**：C. 3
</BlurredAnswer>

### 数列构造

5位员工平均90分，最低77分，分数互不相同，第二名员工至少得多少分？

- A. 90
- B. 91
- C. 92
- D. 93

<BlurredAnswer>
**解题步骤**：

**1. 排序定位**：设第二名最少为x分

**2. 反向构造**：求第二名最小值，让其他分数尽可能大
- 第一名最高99分
- 第三至第五名依次为x-1，x-2，77

**3. 加和求解**：列方程
$$99 + x + (x-1) + (x-2) + 77 = 90×5$$
$$3x + 173 = 450$$
$$x = 92$$

**4. 验证**：分数分配为99, 92, 91, 90, 77，平均分90，符合条件

**答案**：C. 92
</BlurredAnswer>

### 最不利构造

花市现场有郁金香、月季、牡丹各20盆，至少搬出多少盆才能保证一定有郁金香？

- A. 40
- B. 41
- C. 42
- D. 43

<BlurredAnswer>
**解题步骤**：

**1. 识别题型**：题干含"至少…保证…"，属于最不利构造问题

**2. 分析最不利情况**：最不利情况是先搬完所有非郁金香
- 月季20盆 + 牡丹20盆 = 40盆

**3. 构造答案**：答案 = 最不利情况数 + 1 = 40 + 1 = 41盆

**答案**：B. 41
</BlurredAnswer>

### 数列构造

100名学生参加5项活动，人数最多的活动不超过最少的2倍，参加人数最少的活动至少多少人？

- A. 11
- B. 12
- C. 13
- D. 14

<BlurredAnswer>
**解题步骤**：

**1. 排序定位**：设最少活动人数为x，则最多为2x

**2. 反向构造**：求最少活动人数的最小值，让其他活动人数尽可能多
- 其他活动人数最多均为2x

**3. 加和求解**：列方程
$$2x×4 + x = 100$$
$$9x = 100$$
$$x ≈ 11.11$$

**4. 取整规则**：求最小值，向上取整：$$x = 12$$

**答案**：B. 12
</BlurredAnswer>

### 和定最值问题

某单位的五个处室分别有12、18、20、22、30名工作人员，现在要从中抽调45人组成三个工作组，要求每个处室至少抽调2人，每个工作组的人数不得少于10人。问第一处最多抽调多少人？

- A. 8
- B. 9
- C. 10
- D. 11

<BlurredAnswer>
**解题步骤**：

**1. 排序定位**：目标是求第一处抽调人数的最大值

**2. 反向构造**：求第一处最大值，让其他四处抽调人数尽可能小
- 其他四处最少各抽调2人，合计：$$4 × 2 = 8$$人

**3. 加和求解**：理论上第一处最多抽调：$$45 - 8 = 37$$人
- 但第一处总共只有12人，所以实际上限是12人

**4. 验证选项**：用代入排除法验证D选项（11人）
- 其他四处需抽调：$$45 - 11 = 34$$人
- 可行分配：第二处18人，第三处10人，第四处4人，第五处2人
- 验证：$$18 + 10 + 4 + 2 = 34$$，满足条件

**答案**：D. 11
</BlurredAnswer>

## 三、技巧总结

### 1. 题型识别技巧

- 看到<span style={{color:'red'}}>"至少…保证…"</span> → 最不利构造
- 看到<span style={{color:'red'}}>"第几名…最…"</span> → 数列构造
- 看到<span style={{color:'red'}}>"最…最…"</span> → 数列构造  
- 看到<span style={{color:'red'}}>"全部满足…至少"</span> → 多集合反向构造

### 2. 核心解题策略

**极端思维法则**：
- <span style={{color:'red'}}>求最大值 → 其他量取最小值</span>
- <span style={{color:'red'}}>求最小值 → 其他量取最大值</span>

**关键取整规则**：
- <span style={{color:'red'}}>求最小值时向上取整</span>
- <span style={{color:'red'}}>求最大值时向下取整</span>

### 3. 解题步骤标准化

**数列构造问题四步法**：
1. **排序定位**：设所求量为x
2. **反向构造**：根据求最大值还是最小值确定其他量的取值方向
3. **加和求解**：根据总和列方程求解
4. **取整规则**：根据求最大值还是最小值确定取整方向

### 4. 常见易错点

1. **忽略约束条件**：<span style={{color:'orange'}}>注意题目中的所有限制条件，包括隐含条件</span>
2. **取整方向错误**：<span style={{color:'orange'}}>求最小值向上取整，求最大值向下取整</span>
3. **构造不当**：<span style={{color:'orange'}}>没有真正让相关量达到极端值</span>
4. **验证不充分**：<span style={{color:'orange'}}>构造的数值要满足所有约束条件</span>

### 5. 解题技巧补充

1. **代入排除法**：当正面求解困难时，从选项入手验证
2. **数字特性法**：利用奇偶性、倍数关系等特性简化计算
3. **分情况讨论**：复杂问题要分情况分析，确保不遗漏
4. **边界检验**：验证极端情况的合理性和可行性
