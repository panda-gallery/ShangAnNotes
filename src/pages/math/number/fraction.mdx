# 分数数列

## 一、核心概念

分数数列是公务员考试中数字推理部分的常见题型。它的核心特征是数列中包含一个或多个分数。别看它形式复杂，实际上是“纸老虎”。解题的关键在于<span style={{color:'red'}}>将分子和分母拆开，分别作为独立的数列进行分析</span>，或者<span style={{color:'red'}}>将相邻的项进行运算（如交叉相乘、作差、作和）来寻找规律</span>。

常见的解题方法有以下四种：

1.  **分组规律型（分子分母各自成规律）**
    *   **描述**：这是最基础也是最常见的类型。分子组成一个独立的数列，分母也组成一个独立的数列，两者分别遵循简单的规律（如等差、等比、平方、立方等）。
    *   **思路**：分别观察分子和分母的规律。

2.  **交叉影响型**
    *   **描述**：数列中某一项的分子或分母与它相邻项的分子或分母存在某种关联。最经典的就是“前一项的分母是后一项的分子”或者“前一项的分子与分母之和（或差）等于后一项的某个部分”。
    *   **思路**：寻找前后项之间“跨位置”的联系。

3.  **通分/约分找规律**
    *   **描述**：当分数的分子或分母不一致，但可以很简单地化为一致时（通分），或者可以将整数、小数看作特殊的分数时，隐藏的规律就会浮现。
    *   **思路**：将数列中所有项（包括整数）都化成分数，并尝试将分子或分母统一，观察另一部分的变化规律。

4.  **反向约分型（反约分/扩倍）**
    *   **描述**：当所有方法都失效时，这通常是压轴技巧。它与通分相反，需要将某个看似简单的分数进行“扩倍”（分子分母同乘以一个数），以匹配整个数列的规律。
    *   **思路**：<span style={{color:'orange'}}>当其他规律不明显，且数列中某个分数的分子、分母数值特别小时，可以尝试反向约分。</span>

## 二、真题讲解

### 主题1：交叉影响型

**例1**：$\frac{5}{6}$、$\frac{6}{11}$、$\frac{11}{17}$、（）、$\frac{28}{45}$

- A. 17/24
- B. 17/28
- C. 28/17
- D. 24/17

<BlurredAnswer>
**方法：交叉递推**

1.  **第一步：观察数列结构**
    数列由分数组成，且后一项的分子与前一项的分母相同（$6$和$6$，$11$和$11$）。这是典型的交叉影响型特征。

2.  **第二步：确定分子规律**
    根据“<span style={{color:'red'}}>前一项的分母 = 后一项的分子</span>”的规律，未知项的分子应为前一项（$11/17$）的分母，即$17$。

3.  **第三步：确定分母规律**
    观察分母 $6, 11, 17, ?, 45$。我们发现，分母似乎与前一项的分子和分母有关。
    - $11 = 5 + 6$ (第一项的分子+分母)
    - $17 = 6 + 11$ (第二项的分子+分母)
    - 遵循此规律，未知项的分母应为 $11 + 17 = 28$。

4.  **第四步：验证规律**
    我们得到的未知项是 $\frac{17}{28}$。用这个结果来验证下一项：
    - 分子：$28$ (等于 $\frac{17}{28}$ 的分母，正确)
    - 分母：$17 + 28 = 45$ (等于$\frac{17}{28}$的分子+分母，正确)
    规律完全吻合。

因此，答案为 **B. 17/28**。
</BlurredAnswer>

**例2**：$1$、$\frac{2}{3}$、$\frac{5}{8}$、$\frac{13}{21}$、（）

- A. $21/33$
- B. $35/64$
- C. $41/70$
- D. $34/55$

<BlurredAnswer>
**方法：递推求和**

1.  **第一步：统一格式**
    将整数 $1$ 转化为分数 $\frac{1}{1}$，得到新数列：$\frac{1}{1}$、$\frac{2}{3}$、$\frac{5}{8}$、$\frac{13}{21}$、（）。

2.  **第二步：寻找递推关系**
    这是一个更复杂的交叉影响变种。我们尝试寻找前后项分子、分母之间的和差关系。
    - **分子规律**：$1 + 1 = 2$；$2 + 3 = 5$；$5 + 8 = 13$。规律是“<span style={{color:'red'}}>后一项的分子 = 前一项的分子 + 前一项的分母</span>”。
    - **分母规律**：$1 + 2 = 3$；$3 + 5 = 8$；$8 + 13 = 21$。规律是“<span style={{color:'red'}}>后一项的分母 = 前一项的分母 + 后一项的分子</span>”。

3.  **第三步：计算未知项**
    - 未知项的分子：$13 + 21 = 34$。
    - 未知项的分母：$21 + 34 = 55$。
    - 所以未知项为 $\frac{34}{55}$。

因此，答案为 **D. 34/55**。
</BlurredAnswer>

### 主题2：通分/约分找规律

**例3**： $\frac{1}{6}$、$\frac{2}{3}$、$\frac{3}{2}$、$\frac{8}{3}$、（）

- A. $10/3$
- B. $25/6$
- C. $5$
- D. $10$

<BlurredAnswer>
**方法：通分法**

1.  **第一步：观察数列特征**
    数列中各分母 $6, 3, 2, 3$ 有一个明显的公分母 $6$。这提示我们应该使用通分法。

2.  **第二步：执行通分**
    将所有项的分母统一为 $6$：
    - $\frac{1}{6}$ -> $\frac{1}{6}$
    - $\frac{2}{3}$ -> $\frac{4}{6}$
    - $\frac{3}{2}$ -> $\frac{9}{6}$
    - $\frac{8}{3}$ -> $\frac{16}{6}$
    得到新数列：$\frac{1}{6}, \frac{4}{6}, \frac{9}{6}, \frac{16}{6}, ()$。

3.  **第三步：分析新数列规律**
    - 分母：全部为 $6$，是一个常数数列。
    - 分子：$1, 4, 9, 16$。这是非常明显的<span style={{color:'red'}}>平方数列</span>（$1²$, $2²$, $3²$, $4²$）。

4.  **第四步：计算未知项**
    - 分子的下一项是 $5² = 25$。
    - 分母保持 $6$。
    - 未知项为 $\frac{25}{6}$。

因此，答案为 **B. 25/6**。
</BlurredAnswer>

**例5**：$0$、$\frac{1}{3}$、$\frac{1}{2}$、$\frac{3}{5}$、（）

- A. $2/3$
- B. $3/4$
- C. $5/7$
- D. $1$

<BlurredAnswer>
**方法：特殊值转化+各自成规律**

1.  **第一步：处理特殊值**
    数列中有整数$0$和非标准分数$1/2$。我们需要将它们转化为与数列其他项结构相似的分数。
    - 观察分母$3$和$5$，推测$\frac{1}{2}$可以转化为$\frac{2}{4}$。
    - $0$可以看作分子为$0$的分数，为了匹配分母$3,4,5$的规律，将$0$看作$\frac{0}{2}$。

2.  **第二步：形成新数列**
    转化后得到新数列：$\frac{0}{2}$, $\frac{1}{3}$, $\frac{2}{4}$, $\frac{3}{5}$, ()。

3.  **第三步：分析规律**
    这是一个非常清晰的分子分母各自成规律的数列。
    - **分子**：$0, 1, 2, 3$，是公差为$1$的等差数列，下一项是$4$。
    - **分母**：$2, 3, 4, 5$，是公差为$1$的等差数列，下一项是$6$。

4.  **第四步：计算结果**
    - 未知项为 $\frac{4}{6}$，约分后等于 $\frac{2}{3}$。

因此，答案为 **A. 2/3**。
</BlurredAnswer>

### 主题3：反向约分型（扩倍法）

**例4**：$\frac{1}{16}$、$\frac{2}{13}$、$\frac{2}{5}$、$\frac{8}{7}$、$4$、（）

- A. $19/3$
- B. $8$
- C. $16$
- D. $32$

<BlurredAnswer>
**方法：反约分/扩倍法**

1.  **第一步：初步观察与转化**
    将整数$4$看作$\frac{4}{1}$。数列变为 $\frac{1}{16}, \frac{2}{13}, \frac{2}{5}, \frac{8}{7}, \frac{4}{1}, ()$。直接观察分子 $1,2,2,8,4$ 和分母 $16,13,5,7,1$，规律不明显。交叉、作和等也无效。这时要考虑<span style={{color:'orange'}}>反向约分</span>。

2.  **第二步：寻找规律线索**
    观察分子 $1, 2, ..., 8, 4$。我们发现 $1, 2, 8, 4$ 非常接近 $1, 2, 4, 8, 16$ 这样的等比数列。这给了我们一个突破口：尝试将分子构造成一个公比为2的等比数列。

3.  **第三步：执行反约分**
    - 目标分子数列：$1, 2, 4, 8, 16$
    - $\frac{1}{16}$ -> 不变，分子是$1$ ($2⁰$)。
    - $\frac{2}{13}$ -> 不变，分子是$2$ ($2¹$)。
    - $\frac{2}{5}$ -> 为了让分子变成$4$($2²$)，上下同乘$2$，变为 $\frac{4}{10}$。
    - $\frac{8}{7}$ -> 不变，分子是$8$ ($2³$)。
    - $\frac{4}{1}$ -> 为了让分子变成$16$($2⁴$)，上下同乘$4$，变为 $\frac{16}{4}$。

4.  **第四步：分析新数列规律**
    新数列为：$\frac{1}{16}$, $\frac{2}{13}$, $\frac{4}{10}$, $\frac{8}{7}$, $\frac{16}{4}$, ()。
    - **分子**：$1, 2, 4, 8, 16$，是公比为$2$的等比数列，下一项是$32$。
    - **分母**：$16, 13, 10, 7, 4$，是公差为$-3$的等差数列，下一项是 $4 - 3 = 1$。

5.  **第五步：计算结果**
    未知项为 $\frac{32}{1}$，即$32$。

因此，答案为 **D. 32**。
</BlurredAnswer>

### 主题4：作商/作差找规律

**例6**：$1200$，$200$，$40$，（），$\frac{10}{3}$

- A. 10
- B. 20
- C. 30
- D. 5

<BlurredAnswer>
**方法：相邻项作商**

1.  **第一步：观察数列特征**
    数列中各项数值递减，且幅度很大，这通常是作商（或作差）的信号。

2.  **第二步：执行作商**
    - $1200 / 200 = 6$
    - $200 / 40 = 5$

3.  **第三步：发现规律**
    得到的商 $6, 5$ 是一个递减的自然数序列。我们大胆预测，接下来的商应该是 $4$ 和 $3$。

4.  **第四步：计算未知项**
    - $40 / (未知项) = 4$，解得未知项为 $10$。

5.  **第五步：验证规律**
    - $(未知项) / (\frac{10}{3}) = 10 / (\frac{10}{3}) = 10 * (\frac{3}{10}) = 3$。
    - 商的规律 $6, 5, 4, 3$ 完美成立。

因此，答案为 **A. 10**。
</BlurredAnswer>

## 三、技巧总结

1.  **先拆分，后组合**：拿到分数数列，第一反应永远是把分子、分母拆开，看成两个独立的数列，这是最常见的考法。
2.  **整数分数化**：遇到整数或小数，一定要将它转化为分母为1或与数列其他项有联系的分数形式。
3.  **寻找交叉关系**：如果分子分母各自不成规律，立刻检查“前一项分母 -> 后一项分子”或“前项和 -> 后项”等交叉关系。
4.  **大胆通分**：看到分母有公倍数，不要犹豫，立即通分，规律往往就藏在新数列的分子里。
5.  **反向约分是“大招”**：当前三种方法都无效，且数列中有“小”分数时，要想到可能是“伪装”的，需要通过扩倍来“卸妆”。
6.  **作商作差是备选项**：对于包含整数且变化幅度大的数列，作商作差也是一个重要的检查方向。

## 四、巩固练习

**练1**：$\frac{1}{2}$，$\frac{1}{2}$，$\frac{3}{8}$，$\frac{1}{4}$，$\frac{5}{32}$，（）

- A.3/32
- B.1/8
- C.1/6
- D.7/32

<BlurredAnswer>
**提示**：将$\frac{1}{2}$和$\frac{1}{4}$进行通分，使所有项分母统一为$32$。
新数列：$\frac{16}{32}$, $\frac{16}{32}$, $\frac{12}{32}$, $\frac{8}{32}$, $\frac{5}{32}$, ()。
观察分子 $16, 16, 12, 8, 5$，作差得到 $0, -4, -4, -3$... 规律不明显，换个思路。

**正确思路：** 奇数项和偶数项分开看（交叉数列）。
- 奇数项：$\frac{1}{2}, \frac{3}{8}, \frac{5}{32}$。分子是$1,3,5$（等差），分母是$2,8,32$（公比为4的等比）。
- 偶数项：$\frac{1}{2}, \frac{1}{4}, ()$。这是一个公比为$\frac{1}{2}$的等比数列，所以下一项是 $\frac{1}{4} * \frac{1}{2} = \frac{1}{8}$。
答案为 **B. 1/8**。
</BlurredAnswer>

**练2**：$1$，$\frac{2}{3}$，$\frac{1}{2}$，$\frac{2}{5}$，$\frac{1}{3}$、（）

- A.2/7
- B.1/4
- C.5/2
- D.1

<BlurredAnswer>
**提示**：方法一：奇偶项分组。方法二：将所有项通分后观察分子。
**方法二（更优）**：将$1$变成$\frac{2}{2}$，$\frac{1}{2}$变成$\frac{2}{4}$，$\frac{1}{3}$变成$\frac{2}{6}$。
新数列：$\frac{2}{2}, \frac{2}{3}, \frac{2}{4}, \frac{2}{5}, \frac{2}{6}, ()$。
分子为常数$2$，分母为 $2,3,4,5,6$ 的等差数列，下一项是$7$。
所以未知项是 $\frac{2}{7}$。
答案为 **A. 2/7**。
</BlurredAnswer>
