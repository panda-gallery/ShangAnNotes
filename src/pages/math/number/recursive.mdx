# 第六课 递推数列

## 一、核心概念

### 1. 什么是递推数列？

想象一下多米诺骨牌，推倒第一张，后面的骨牌会依次倒下。每一张骨牌倒下的状态，都由它前面的骨牌决定。


**递推数列 (Recursive Sequence)** 的概念与此类似：<span style={{color:'red'}}>数列中的某一项，可以通过它前面的一项或几项，按照一个固定的规则计算出来</span>。这个固定的规则，我们称之为 **递推关系**。

例如，著名的斐波那契数列 `1, 1, 2, 3, 5, 8, ...`，从第三项开始，每一项都等于前两项之和。
- 第3项 `2` = 第1项 `1` + 第2项 `1`
- 第4项 `3` = 第2项 `1` + 第3项 `2`
- ...

这个“等于前两项之和”就是它的递推关系。如果我们用数学语言表达，就是 $a_n = a_{n-1} + a_{n-2}$。

在公务员考试中，递推数列的变化形式多样，但万变不离其宗，核心都是<span style={{color:'red'}}>找出这个隐藏在数字背后的“递推关系”</span>。常见的递推关系有以下几种：

- **和/差/积/商递推**：当前项由前两项通过简单的加减乘除得到。
- **平方/立方递推**：当前项与前一项的平方或立方有关。
- **多级递推与修正**：有时，简单的递推关系不成立，需要加上或减去一个变化的“修正项”（可能是一个常数，也可能是一个变化的数列）。

### 2. 核心公式推导

让我们通过一些生活中的例子，来理解这些递推关系是如何构建的。

#### 示例：电影院的座位安排

假设一个电影院，为了让观众感觉更宽敞，设计了一种特殊的座位安排：
- 第1排有 `2` 个座位。
- 第2排有 `3` 个座位。
- 从第3排开始，每一排的座位数都是<span style={{color:'red'}}>前一排座位数的平方，再减去前两排的座位数之和</span>。

请问第4排有多少个座位？

**分析与推导**：
这是一个典型的递推问题。我们来一步步拆解：

1.  **确定已知项**：
    - 第1排座位数 $a_1 = 2$
    - 第2排座位数 $a_2 = 3$

2.  **找出递推关系**：
    - 文字描述：“前一排座位数的平方，再减去前两排的座位数之和”
    - 翻译成数学公式：$a_n = (a_{n-1})^2 - (a_{n-1} + a_{n-2})$

3.  **计算第3排座位数 ($a_3$)**：
    - 根据公式，$a_3 = (a_2)^2 - (a_2 + a_1)$
    - 代入数值：$a_3 = 3^2 - (3 + 2) = 9 - 5 = 4$
    - 所以，第3排有4个座位。

4.  **计算第4排座位数 ($a_4$)**：
    - 根据公式，$a_4 = (a_3)^2 - (a_3 + a_2)$
    - 代入数值：$a_4 = 4^2 - (4 + 3) = 16 - 7 = 9$
    - 所以，第4排有9个座位。

通过这个例子，我们不仅理解了递推关系，还亲自推导并求解了一个递推数列。考试中的题目本质上就是逆向操作：<span style={{color:'orange'}}>给你一串数字，让你反推出这个“座位安排”的规则</span>。

## 二、真题讲解

掌握了核心概念后，我们来看几道真题，学习如何在实战中应用。我们的核心策略是：**看趋势，做试探**。

- **看趋势**：观察数列数字的增减趋势。是快速变大（可能和乘法、平方有关）？还是平稳变化（可能和加减有关）？或是有正有负（可能涉及减法）？
- **做试探**：根据趋势，大胆猜测一个递推关系，并从前往后验证。一次不行，就换一种思路。

### 类型一：和差类递推

**例1**：` ( ), 47, 36, 83, 119`

- A. -37
- B. -11
- C. 11
- D. 37

<BlurredAnswer>
**思路剖析：**

1.  **观察趋势**：数列数字有增有减，但整体波动不大，优先考虑加减法关系。
2.  **做试探（和关系）**：我们尝试从后往前验证，看是否存在 $a_{n-2} + a_{n-1} = a_n$ 的关系。
    - 验证第4、5项：$36 + 83 = 119$。符合！
    - 验证第3、4项：$47 + 36 = 83$。符合！
3.  **确定规律**：递推关系为 <span style={{color:'red'}}>$a_n = a_{n-2} + a_{n-1}$</span>（斐波那契数列的变种）。
4.  **求解未知项**：设未知项为 $x$。根据规律，$x + 47 = 36$。
5.  **计算结果**：$x = 36 - 47 = -11$。

因此，答案为 B。
</BlurredAnswer>

### 类型二：积商类递推（带修正）

**例2**： `2, 4, 3, 7, 16, 107, ( )`

- A. 1594
- B. 1684
- C. 1707
- D. 1856

<BlurredAnswer>
**思路剖析：**

1.  **观察趋势**：数列从 `16` 到 `107` 增长迅猛，暗示可能存在乘法或平方关系。
2.  **做试探（积关系）**：我们尝试用前两项的积来表示后一项。
    - 试探 $a_3$：$a_1 \times a_2 = 2 \times 4 = 8$。而 $a_3=3$。两者相差 `5`。即 $2 \times 4 - 5 = 3$。
    - 试探 $a_4$：$a_2 \times a_3 = 4 \times 3 = 12$。而 $a_4=7$。两者相差 `5`。即 $4 \times 3 - 5 = 7$。
    - 试探 $a_5$：$a_3 \times a_4 = 3 \times 7 = 21$。而 $a_5=16$。两者相差 `5`。即 $3 \times 7 - 5 = 16$。
3.  **确定规律**：递推关系为 <span style={{color:'red'}}>$a_n = a_{n-2} \times a_{n-1} - 5$</span>。这是一个带常数修正的积数列。
4.  **验证规律**：$a_4 \times a_5 - 5 = 7 \times 16 - 5 = 112 - 5 = 107$。$a_6$ 符合规律。
5.  **求解未知项**：未知项 $a_7 = a_5 \times a_6 - 5 = 16 \times 107 - 5$。
6.  **计算结果**：$16 \times 107 - 5 = 1712 - 5 = 1707$。

因此，答案为 C。
</BlurredAnswer>

### 类型三：平方类递推（带修正）

**例3**：`2, 3, 7, 45, 2017, ( )`

- A. 4068271
- B. 4068273
- C. 4068275
- D. 4068277

<BlurredAnswer>
**思路剖析：**

1.  **观察趋势**：数列数字增长速度极快（$45 \rightarrow 2017$），强烈暗示平方关系。
2.  **做试探（平方关系）**：我们尝试用前一项的平方来表示后一项。
    - 试探 $a_2$：$a_1^2 = 2^2 = 4$。而 $a_2 = 3$。两者相差 `1`。即 $2^2 - 1 = 3$。
    - 试探 $a_3$：$a_2^2 = 3^2 = 9$。而 $a_3 = 7$。两者相差 `2`。即 $3^2 - 2 = 7$。
    - 试探 $a_4$：$a_3^2 = 7^2 = 49$。而 $a_4 = 45$。两者相差 `4`。即 $7^2 - 4 = 45$。
3.  **寻找修正项规律**：修正项（减去的数）分别为 `1, 2, 4`。这是一个公比为2的等比数列。
4.  **确定规律**：递推关系为 <span style={{color:'red'}}>$a_n = (a_{n-1})^2 - 2^{n-2}$</span>。
5.  **验证规律**：$a_5 = a_4^2 - 2^{5-2} = 45^2 - 2^3 = 2025 - 8 = 2017$。$a_5$ 符合规律。
6.  **求解未知项**：未知项 $a_6 = a_5^2 - 2^{6-2} = 2017^2 - 2^4 = 2017^2 - 16$。
7.  **计算结果**：$2017^2$ 的尾数是 $7 \times 7 = 9$。$2017^2 - 16$ 的尾数是 $9-6=3$。观察选项，只有B的尾数是3。

因此，答案为 B。
</BlurredAnswer>

### 类型四：复杂积商递推

**例4**：`3, 2, 4, 4, 12, 36, ( )`

- A. 264
- B. 396
- C. 480
- D. 600

<BlurredAnswer>
**思路剖析：**

1.  **观察趋势**：数列有增有减，且增长速度较快，可以同时考虑加减和乘除。
2.  **做试探**：这类题目规律较为隐藏，需要更大胆的尝试。
    - 尝试 `和/差/积` 的简单组合，会发现走不通。
    - 尝试更复杂的组合，例如 $(a_{n-2} \pm C) \times a_{n-1}$。
    - 我们试探 $(a_{n-2} - 1) \times a_{n-1}$：
        - $a_3 = (a_1 - 1) \times a_2 = (3 - 1) \times 2 = 4$。符合！
        - $a_4 = (a_2 - 1) \times a_3 = (2 - 1) \times 4 = 4$。符合！
        - $a_5 = (a_3 - 1) \times a_4 = (4 - 1) \times 4 = 12$。符合！
3.  **确定规律**：递推关系为 <span style={{color:'red'}}>$a_n = (a_{n-2} - 1) \times a_{n-1}$</span>。
4.  **验证规律**：$a_6 = (a_4 - 1) \times a_5 = (4 - 1) \times 12 = 36$。$a_6$ 符合规律。
5.  **求解未知项**：未知项 $a_7 = (a_5 - 1) \times a_6 = (12 - 1) \times 36$。
6.  **计算结果**：$11 \times 36 = 396$。

因此，答案为 B。
</BlurredAnswer>

## 三、技巧总结

1.  **趋势优先**：
    - <span style={{color:'orange'}}>变化平缓</span> -> 优先考虑 **和、差** 关系。
    - <span style={{color:'orange'}}>急剧增长</span> -> 优先考虑 **积、平方** 关系。
    - <span style={{color:'orange'}}>有正有负/忽大忽小</span> -> 优先考虑 **减法** 或 **带负号的乘法**。

2.  **大胆试探，小心验证**：
    - 从最简单的 `和差积商` 开始试探。
    - 如果不符合，尝试引入 `修正项`（常数修正、序列修正）。
    - 始终从前往后，至少验证2-3项来确认你的猜想。

3.  **尾数法**：
    - 当计算量巨大，但选项尾数各不相同时，果断使用尾数法秒杀，如【例3】。

4.  **记住常见模型**：
    - **和/差数列**：$a_n = a_{n-1} + a_{n-2}$ 或 $a_n = a_{n-1} - a_{n-2}$
    - **积/商数列**：$a_n = a_{n-1} \times a_{n-2}$
    - **平方数列**：$a_n = (a_{n-1})^2 + C$
    - 掌握这些基础模型，考场上就能更快识别出变体。

递推数列的魅力在于它的千变万化，但其解题的钥匙始终是<span style={{color:'red'}}>“观察-猜想-验证”</span>这一逻辑闭环。多加练习，培养对数字的敏感度，你一定能攻克它！
