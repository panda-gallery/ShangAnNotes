# 约数和余数

约数和余数问题是数量关系中的基础模块，但其出题形式灵活多变，常常与其他知识点结合，是考试中的常客。掌握其核心原理和解题技巧至关重要。

## 核心概念

### 1. 除法基本公式

<span style={{color:'red'}}>若整数 a 能被整数 b(b≠0) 整除 (即 a÷b 余数为 0)，则称 b 是 a 的约数。</span>

我们先从一个生活中的小例子开始。假设班上有13颗糖果，要平均分给4位同学，每位同学最多能分到几颗？还剩下几颗？

很简单，我们用除法：$13 \div 4 = 3 \dots\dots 1$

- **被除数**：13 (等待被分配的糖果总数)
- **除数**：4 (分配对象的数量，即同学人数)
- **商**：3 (每位同学分到的糖果数)
- **余数**：1 (分配后剩余的糖果数)

从这个例子中，我们可以清晰地看到它们之间的关系：`13颗糖果 = 4位同学 × 每人3颗 + 剩下的1颗`。

由此，我们推导出除法中最核心、最基本的公式：

<span style={{color:'red'}}>**被除数 = 除数 × 商 + 余数**</span>

这个公式是解决几乎所有余数问题的基石。

同时，我们注意到，剩下的糖果（余数）数量必须比同学的人数（除数）要少，否则我们就可以再给每位同学分一颗。因此，我们得到余数的一个关键性质：

<span style={{color:'red'}}>**0 ≤ 余数 < 除数**</span>

<span style={{color:'orange'}}>易错点：余数不能为负数，且必须小于除数。</span>

### 2. 余数的核心运算性质

余数在加、减、乘法运算中具有非常稳定和优美的性质，这对于简化计算至关重要。

-   **可加性**：和的余数等于余数之和的余数。
    -   公式表达：$(a + b) \pmod m = ((a \pmod m) + (b \pmod m)) \pmod m$
-   **可减性**：差的余数等于余数之差的余数。
    -   公式表达：$(a - b) \pmod m = ((a \pmod m) - (b \pmod m)) \pmod m$
-   **可乘性**：积的余数等于余数之积的余数。
    -   公式表达：$(a \times b) \pmod m = ((a \pmod m) \times (b \pmod m)) \pmod m$

<span style={{color:'orange'}}>这里的 `a mod m` 表示 `a` 除以 `m` 的余数。这个性质可以帮助我们处理大数计算问题，只需对各个部分取余后再进行计算。</span>

### 3. 核心解题模型

在掌握了基本公式后，我们可以进一步探讨三种在考试中非常常见的模型。

#### a) 同余问题 (余数相同)

**场景**: 一个数 $N$ 分别除以好几个不同的数（例如 $a, b, c$），得到的余数都是同一个数 $r$。

**推导**:
根据核心公式，我们可以得到：
- $N = a \times \text{商}_1 + r \implies N - r = a \times \text{商}_1$
- $N = b \times \text{商}_2 + r \implies N - r = b \times \text{商}_2$
- $N = c \times \text{商}_3 + r \implies N - r = c \times \text{商}_3$

观察上面的式子，我们发现 $N-r$ 这个整体，既是 $a$ 的倍数，也是 $b$ 的倍数，还是 $c$ 的倍数。在数学上，我们就称 $N-r$ 是 $a, b, c$ 的 **公倍数**。

**结论**:
<span style={{color:'red'}}>如果一个数除以 $a,b,c$ 都余 $r$，那么这个数可以表示为：$N = k \times \text{LCM}(a, b, c) + r$，其中 $\text{LCM}(a, b, c)$ 表示 $a, b, c$ 的最小公倍数，$k$ 为非负整数。</span>

#### b) 同亏问题 (除数与余数的差相同)

**场景**: 一个数 $N$ 分别除以 $a, b, c$，得到的余数不尽相同，但是“除数 - 余数”的结果都等于一个固定的数 $d$。这种情况我们称之为“同亏”，意思是“都缺少 $d$ 才能被整除”。

**推导**:
根据题意，我们可以得到：
- $N$ 除以 $a$，余数为 $a-d$。根据核心公式：$N = a \times \text{商}_1 + (a-d)$。
  我们把公式变个形：$N+d = a \times \text{商}_1 + a = a \times (\text{商}_1+1)$。
- $N$ 除以 $b$，余数为 $b-d$。根据核心公式：$N = b \times \text{商}_2 + (b-d)$。
  变形后：$N+d = b \times \text{商}_2 + b = b \times (\text{商}_2+1)$。
- $N$ 除以 $c$，余数为 $c-d$。根据核心公式：$N = c \times \text{商}_3 + (c-d)$。
  变形后：$N+d = c \times \text{商}_3 + c = c \times (\text{商}_3+1)$。

观察变形后的式子，我们发现 $N+d$ 这个整体，同时是 $a, b, c$ 的公倍数。

**结论**:
<span style={{color:'red'}}>如果一个数除以 $a$ 余 $a-d$，除以 $b$ 余 $b-d$，除以 $c$ 余 $c-d$，那么这个数可以表示为：$N = k \times \text{LCM}(a, b, c) - d$，其中 $\text{LCM}(a, b, c)$ 表示 $a, b, c$ 的最小公倍数，$k$ 为正整数。</span>

#### c) 和同问题 (除数与余数的和相同)

**场景**: 一个数 $N$ 分别除以 $a, b, c$，得到的 “除数 + 余数” 的结果都等于一个固定的数 $s$。

**推导**:
根据题意，我们可以得到：
- $N$ 除以 $a$，余数为 $s-a$。根据核心公式：$N = a \times \text{商}_1 + (s-a)$。
  我们把公式变个形：$N-s = a \times \text{商}_1 - a = a \times (\text{商}_1-1)$。
- $N$ 除以 $b$，余数为 $s-b$。根据核心公式：$N = b \times \text{商}_2 + (s-b)$。
  变形后：$N-s = b \times \text{商}_2 - b = b \times (\text{商}_2-1)$。

观察变形后的式子，我们发现 $N-s$ 这个整体，同时是 $a, b$ 的公倍数。

**结论**:
<span style={{color:'red'}}>如果一个数除以 $a$ 余 $s-a$，除以 $b$ 余 $s-b$，那么这个数可以表示为：$N = k \times \text{LCM}(a, b) + s$，其中 $\text{LCM}(a, b)$ 表示 $a, b$ 的最小公倍数，$k$ 为非负整数。</span>

## 真题讲解

### 主题1：基础余数问题

这类题目直接考察对核心公式 `被除数 = 除数 × 商 + 余数` 的理解和应用。

**例1**：
一个正整数P，满足 P 除以 7 的商是 21，余数是 4。请问P的值是多少？

- A. 141
- B. 147
- C. 151
- D. 155

<BlurredAnswer>
**思路剖析**：

1.  **第一步：识别题型**
    题目中明确给出了除数、商和余数，要求被除数。这是最基础的余数问题，直接套用核心公式即可。

2.  **第二步：代入公式**
    - 被除数 = P
    - 除数 = 7
    - 商 = 21
    - 余数 = 4

    根据公式：`被除数 = 除数 × 商 + 余数`
    $P = 7 \times 21 + 4$

3.  **第三步：计算结果**
    $P = 147 + 4 = 151$

因此，答案为 C. 151
</BlurredAnswer>

### 主题2：同余问题

**例2**：（2013天津公务员）
学生列队，人数在90到110人之间。若排成5排则少2人，排成7排则少4人。求学生人数？

- A. 98
- B. 103
- C. 108
- D. 109

<BlurredAnswer>
**思路剖析**：

1.  **第一步：转化题意，识别题型**
    设学生总人数为 $N$。
    - “排成5排则少2人”意味着 $N$ 除以5，还差2个才能够被整除。如果队伍再来2个人，就能被5整除，即 $N+2$ 是5的倍数。或者可以理解为 $N$ 除以5余 $5-2=3$ 人。
    - “排成7排则少4人”意味着 $N$ 除以7，还差4个才能够被整除。如果队伍再来4个人，就能被7整除，即 $N+4$ 是7的倍数。或者可以理解为 $N$ 除以7余 $7-4=3$ 人。

    <span style={{color:'orange'}}>这里需要一个思维转换：“排成 a 排少 b 人”等价于“总人数除以 a 余 a-b 人”。</span>
    
    经过转换，两个条件都变成了“除以x余3”，这是典型的 **同余问题**。

2.  **第二步：应用同余结论**
    根据同余问题的结论，该数 $N$ 必须满足 $N = k \times \text{LCM}(5, 7) + 3$。

3.  **第三步：计算最小公倍数 (LCM)**
    5和7互质，它们的最小公倍数就是它们的乘积。
    $\text{LCM}(5, 7) = 5 \times 7 = 35$。

4.  **第四步：构建通项公式并结合范围求解**
    所以，学生人数 $N$ 的通项公式为：$N = 35k + 3$（其中 $k$ 为非负整数）。
    题目要求人数在90到110人之间，我们代入 $k$ 值进行尝试：
    - 当 $k=0$ 时，$N = 35 \times 0 + 3 = 3$。（不符）
    - 当 $k=1$ 时，$N = 35 \times 1 + 3 = 38$。（不符）
    - 当 $k=2$ 时，$N = 35 \times 2 + 3 = 73$。（不符）
    - 当 $k=3$ 时，$N = 35 \times 3 + 3 = 108$。（符合）

因此，答案为 C. 108
</BlurredAnswer>

### 主题3：同亏问题

**例3**：
有一个数，用3除余2，用4除余3，用5除余4。这个数最小是多少？

- A. 57
- B. 58
- C. 59
- D. 60

<BlurredAnswer>
**思路剖析**：

1.  **第一步：识别题型**
    设这个数为 $N$。
    - $N$ 除以3余2，发现 $3-2=1$。
    - $N$ 除以4余3，发现 $4-3=1$。
    - $N$ 除以5余4，发现 $5-4=1$。
    “除数-余数”的差是恒定的值1。这是典型的 **同亏问题**，亏数为1。

2.  **第二步：应用同亏结论**
    根据同亏问题的结论，这个数 $N$ 满足 $N = k \times \text{LCM}(3, 4, 5) - 1$。

3.  **第三步：计算最小公倍数 (LCM)**
    $\text{LCM}(3, 4, 5) = 3 \times 4 \times 5 = 60$。

4.  **第四步：构建通项公式并求解**
    所以，$N$ 的通项公式为 $N = 60k - 1$（其中 $k$ 为正整数）。
    题目要求最小的数，我们取 $k=1$。
    $N = 60 \times 1 - 1 = 59$。

因此，答案为 C. 59
</BlurredAnswer>

### 主题4：复杂余数问题（中国剩余定理）

当余数没有明显规律（既不同余也不同亏）时，我们需要使用更通用的方法，通常称为“逐级满足法”或“步步推进法”。

**例4**（孙子算经原题）：
今有物不知其数，三三数之剩二，五五数之剩三，七七数之剩二，问物几何？
（一个数除以3余2，除以5余3，除以7余2，求这个数最小是多少？）

- A. 23
- B. 28
- C. 33
- D. 38

<BlurredAnswer>
**思路剖析**：

这种问题没有简单的“同余”或“同亏”规律，我们需要一步步来限制这个未知数。技巧是<span style={{color:'orange'}}>从除数最大的条件开始，逐步代入验证</span>。

1.  **第一步：从最大的除数入手**
    “除以7余2”，我们可以把这个数表示为 $N = 7k + 2$ (k为非负整数)。

2.  **第二步：代入第二个条件**
    这个数还要满足“除以5余3”。我们把 $N=7k+2$ 代入：
    $(7k+2)$ 除以5余3。
    我们可以把 $7k$ 拆成 $5k+2k$，这样 $5k$ 一定能被5整除。
    所以问题转化为 $(2k+2)$ 除以5余3。
    即 $2k$ 除以5余1。
    我们可以口算试一下：
    - k=1, 2k=2, 除以5余2
    - k=2, 2k=4, 除以5余4
    - k=3, 2k=6, 除以5余1。找到了！
    所以 $k$ 的最小值为3。根据余数性质，k的通项是 $k=5j+3$ (j为非负整数)。

3.  **第三步：更新N的表达式**
    将 $k=5j+3$ 代回 $N$ 的表达式：
    $N = 7(5j+3) + 2 = 35j + 21 + 2 = 35j + 23$。
    这个表达式 $N=35j+23$ 已经同时满足“除以5余3”和“除以7余2”两个条件了。

4.  **第四步：代入最后一个条件**
    这个数还要满足“除以3余2”。我们把 $N=35j+23$ 代入：
    $(35j+23)$ 除以3余2。
    $35j$ 可以拆成 $33j+2j$，$23$ 可以拆成 $21+2$。
    所以问题转化为 $(2j+2)$ 除以3余2。
    即 $2j$ 除以3余0，说明 $2j$ 是3的倍数，那么 $j$ 必须是3的倍数。
    $j$ 的最小值为0。

5.  **第五步：求解最终结果**
    要求最小的数，我们取 $j=0$。
    代入 $N = 35j+23$，得 $N = 35 \times 0 + 23 = 23$。

因此，答案为 A. 23
</BlurredAnswer>

### 主题5：反求除数问题

**例5**：
一个数除 300 余 5，除 400 余 10，求该数的最大值。

- A. 5
- B. 15
- C. 59
- D. 65

<BlurredAnswer>
**思路剖析**：

1.  **第一步：识别题型，转化条件**
    这道题不是求被除数，而是反过来求除数。设这个除数为 $d$。
    - “一个数除 300 余 5”可以根据核心公式写成 $300 = d \times \text{商}_1 + 5$。
    - “一个数除 400 余 10”可以写成 $400 = d \times \text{商}_2 + 10$。

2.  **第二步：运用整除关系**
    将上述等式变形：
    - $300 - 5 = 295 = d \times \text{商}_1$。这意味着 $d$ 是 295 的约数。
    - $400 - 10 = 390 = d \times \text{商}_2$。这意味着 $d$ 是 390 的约数。
    
    因此，$d$ 必须是 295 和 390 的 **公约数**。

3.  **第三步：求解公约数**
    我们来分解质因数，找到所有的公约数：
    - $295 = 5 \times 59$
    - $390 = 10 \times 39 = 2 \times 5 \times 3 \times 13$
    295 和 390 的公约数只有 1 和 5。

4.  **第四步：结合余数性质确定最终解**
    核心性质之一是<span style={{color:'red'}}>“余数必须小于除数”</span>。
    - 条件一的余数是 5，所以 $d > 5$。
    - 条件二的余数是 10，所以 $d > 10$。
    综合来看，除数 $d$ 必须大于 10。
    
    我们求出的公约数是 1 和 5，都不满足 $d > 10$ 的条件。这说明我们的计算或理解有误。
    
    <span style={{color:'orange'}}>**重新审题**：题目是“一个数`除`300 余 5”，这里有两种理解：</span>

    1.  $N \div d = 300 \dots 5$ (N是被除数，d是除数)
    2.  $300 \div d = q \dots 5$ (300是被除数，d是除数)
    
    中文语境下，“A除B”表示 B/A，“A除以B”表示 A/B。但考试中为了避免歧义，通常表达会更清晰。按常规理解，应为第二种。让我们按第二种方式重新解题。

    **正确解法**：
    1.  **转化条件**：
        - $300$ 除以 $d$ 余 5 $\implies 300 - 5 = 295$ 能被 $d$ 整除。
        - $400$ 除以 $d$ 余 10 $\implies 400 - 10 = 390$ 能被 $d$ 整除。
    2.  **求解公约数**：
        $d$ 是 295 和 390 的公约数。
        - $295 = 5 \times 59$
        - $390 = 2 \times 3 \times 5 \times 13$
        - 公约数有 1 和 5。
    3.  **验证余数条件**：
        除数必须大于余数。$d > 5$ 且 $d > 10$，所以 $d > 10$。
        公约数 1 和 5 都不满足条件。这说明题目条件可能存在问题或印刷错误。
        
        **让我们检查一下题目本身是否可能有变化。**
        假设题目是“一个数`被`300除余5，`被`400除余10”，这是完全不同的问题。
        假设题目是“300除以一个数余5，400除以这个数余10”，这是我们正在解的。
        
        如果我们将 $295$ 的约数（1, 5, 59, 295）和 $390$ 的约数（1, 2, 3, 5, 6, 10, 13...）进行比较，会发现公约数似乎不只1和5。
        $295 = 5 \times 59$
        $390 = 2 \times 3 \times 5 \times 13$
        最大公约数是 5。公约数只有 1, 5。计算无误。
        
        **结论**：在严格的数学逻辑下，此题无解。但在行测考试中，有时会出现不严谨的题目，需要我们猜测出题人意图。如果我们将 `300` 和 `295` 看作是笔误，例如 `295` 应该是 `2950`，那就复杂了。
        
        **另一种可能：**
        "一个数除300余5"，会不会是 “一个数 N，N 除以 300 余 5”。
        "一个数除400余10"，会不会是 “这个数 N，N 除以 400 余 10”。
        $N = 300k+5$
        $N = 400j+10$
        求 $N$ 的最大值？题目没给范围，最大值是无穷。
        
        **最可能的意图，还是求除数**。如果题目数字有误，比如将 390 改为 395 (400-5)，那么 GCD(295, 395) = 5，还是无解。如果将 295 改为 290(300-10), GCD(290, 390) = 10，d>10还是无解。
        
        **让我们重新检查分解质因数。$295 = 5 \times 59$，$390 = 2 \times 3 \times 5 \times 13$。公约数只有1和5。**
        
        **这道题本身有问题，无法得出选项中的答案。** 为了教学目的，我们修改一下题目：
        **修改后题目**：一个数除 240 余 5，除 300 余 15，求该数的最大值。
        1.  $240-5=235$, $300-15=285$。
        2.  $d$ 是 235 和 285 的公约数，且 $d>15$。
        3.  $235 = 5 \times 47$，$285 = 3 \times 5 \times 19$。
        4.  公约数是1, 5。依然无解。
        
        **让我们直接从答案反推**：如果答案是59(C)。
        $300 = 5 \times 59 + 5$ (正确)
        $400 = 6 \times 59 + 46$ (错误，余数应为10)
        
        这道例题本身存在严重问题。在实际考试中，遇到这种题目，需要标记并跳过，或选择一个看起来最可能的选项（如果必须选）。**此处我们不再深入探讨此题，仅展示“反求除数”这一题型和基本解法思路。**

</BlurredAnswer>

### 主题6：余数性质应用题

**例6**：（2019国考）
商店里有6箱货物，分别重15、16、18、19、20、31千克。两位顾客买走了其中的5箱。已知一位顾客买的货物重量是另一位的2倍。问商店剩下的一箱货物重多少千克？

- A. 16
- B. 18
- C. 19
- D. 20

<BlurredAnswer>
**思路剖析**：

1.  **第一步：识别题型，抓住核心**
    题目涉及重量的加和与倍数关系，可以巧妙地利用“余数”的性质来简化计算。
    核心条件是：一位顾客的重量（$W_1$）是另一位（$W_2$）的2倍，即 $W_1 = 2W_2$。

2.  **第二步：利用余数性质构建等式**
    两位顾客买走的总重量 $W_{total} = W_1 + W_2 = 2W_2 + W_2 = 3W_2$。
    这个等式告诉我们一个关键信息：<span style={{color:'red'}}>两位顾客买走的总重量必须是3的倍数。</span>

3.  **第三步：计算总重量，并对3取余**
    首先，计算所有6箱货物的总重量：
    $15+16+18+19+20+31 = 119$ 千克。

    接下来，我们不需要计算每个数字除以3的余数，而是可以直接计算总和除以3的余数：
    $119 \div 3 = 39 \dots\dots 2$。
    所以，6箱货物的总重量除以3余2。

4.  **第四步：结合“被3整除”的条件反推**
    我们知道，买走的5箱货物的总重量必须是3的倍数（即除以3余0）。
    设剩下的一箱货物重量为 $W_{left}$。
    那么，`（6箱总重量） - （剩下1箱的重量） = （买走5箱的总重量）`
    用余数的语言来表达就是：
    `(6箱总重量 mod 3) - (剩下1箱重量 mod 3) = (买走5箱总重量 mod 3)`
    代入我们已知的值：
    `2 - (W_left mod 3) = 0`
    
    这个方程在余数运算中需要注意，减法可能出现负数。更稳妥的写法是：
    `(6箱总重量 mod 3) = (买走5箱总重量 mod 3 + 剩下1箱重量 mod 3) mod 3`
    `2 = (0 + W_left mod 3) mod 3`
    所以，<span style={{color:'red'}}>剩下那箱货物的重量除以3必须余2。</span>

5.  **第五步：验证选项**
    我们来检查每个选项的重量除以3的余数：
    - A. 16: $16 \div 3$ 余 1
    - B. 18: $18 \div 3$ 余 0
    - C. 19: $19 \div 3$ 余 1
    - D. 20: $20 \div 3$ 余 2
    
    只有20千克满足“除以3余2”的条件。

因此，答案为 D. 20

## 技巧总结

1.  **牢记一个中心**：
    万变不离其宗，所有余数问题都围绕核心公式 <span style={{color:'red'}}>`被除数 = 除数 × 商 + 余数`</span> 展开。

2.  **掌握核心模型与口诀**：
    快速识别“余同”、“差同”、“和同”模型，是高效解题的关键。

| 类型 | 条件 | 通式 | 口诀 |
| :--- | :--- |:--- | :--- |
| **余同** | 余数 $r$ 相同 | $N = \text{LCM} \cdot k + r$ | 最小公倍数加余数 |
| **差同** | 除数与余数之差 $d$ 相同 | $N = \text{LCM} \cdot k - d$ | 最小公倍数减差数 |
| **和同** | 除数与余数之和 $s$ 相同 | $N = \text{LCM} \cdot k + s$ | 最小公倍数加和数 |

3.  **精通一种通用方法**：
    - **“步步推进法”** (解决中国剩余定理问题)：当余数无规律时，这是最有效的方法。
      - **步骤**：从最大的除数开始，设出数的表达式 $\rightarrow$ 代入下一个除数条件，解出参数的最小取值并写出通项 $\rightarrow$ 更新数的表达式 $\rightarrow$ 重复此过程，直到满足所有条件。

4.  **活用两大性质**：
    - **余数范围**：`0 ≤ 余数 < 除数`。这个性质在反求除数的题目中是关键约束条件。
    - **余数运算**：和差积的余数等于余数的和差积。这个性质在处理倍数、加和问题时有奇效。
