# 特殊数字乘法速算技巧

在行测资料分析和数量关系模块中，时间就是分数。掌握一些常用的速算技巧，能够极大地提升解题速度和准确率。本文将重点讲解与1.1、1.5、0.9这三个特殊数字相乘的速算技巧。

## 一、核心概念

特殊数字乘法的核心原理是<span style={{color:'red'}}>乘法分配律</span>。我们通过将特殊数字拆解为整数和简单小数的和或差，然后利用分配律将复杂的乘法运算转化为简单的加减法，从而实现快速心算。

### （一）乘以 1.1 (原数增加10%)

我们来看一个生活中的例子：假设你购买了一件商品，价格为A元，需要额外支付10%的消费税。那么你总共需要支付多少钱呢？

总金额 = 商品价格 + 消费税 <br />
总金额 = A + A × 10% <br />
总金额 = A × (1 + 0.1) <br />
总金额 = A × 1.1 <br/>

通过这个例子，我们推导出乘以1.1的速算公式：

<span style={{color:'red'}}>$$A \times 1.1 = A \times (1 + 0.1) = A \times 1 + A \times 0.1 = A + \frac{A}{10}$$</span>

**操作步骤：**
1.  计算原数A的十分之一（即<span style={{color:'purple'}}>小数点向左移动一位</span>）。
2.  将计算结果与原数A相加。

**示例：** 计算 $84 \times 1.1$
1.  $84$ 的十分之一是 $8.4$。
2.  $84 + 8.4 = 92.4$。

### （二）乘以 1.5 (原数增加50%)

假设你这个月表现出色，老板决定给你发50%的月薪作为奖金。如果你的月薪是A元，那么你这个月总共能拿到多少钱？

总收入 = 基本工资 + 奖金
总收入 = A + A × 50%
总收入 = A × (1 + 0.5)
总收入 = A × 1.5

因此，乘以1.5的速算公式为：

<span style={{color:'red'}}>$$A \times 1.5 = A \times (1 + 0.5) = A \times 1 + A \times 0.5 = A + \frac{A}{2}$$</span>

**操作步骤：**
1.  计算原数A的一半。
2.  将计算结果与原数A相加。

**示例：** 计算 $128 \times 1.5$
1.  $128$ 的一半是 $64$。
2.  $128 + 64 = 192$。

### （三）乘以 0.9 (原数减少10%)

商场大促销，所有商品打九折。你看中一件原价为A元的衣服，打折后需要多少钱？

折后价 = 原价 - 折扣金额
折后价 = A - A × 10%
折后价 = A × (1 - 0.1)
折后价 = A × 0.9

因此，乘以0.9的速算公式为：

<span style={{color:'red'}}>$$A \times 0.9 = A \times (1 - 0.1) = A \times 1 - A \times 0.1 = A - \frac{A}{10}$$</span>

**操作步骤：**
1.  计算原数A的十分之一（<span style={{color:'purple'}}>小数点向左移动一位</span>）。
2.  用原数A减去该结果。

**示例：** 计算 $56 \times 0.9$
1.  $56$ 的十分之一是 $5.6$。
2.  $56 - 5.6 = 50.4$。

## 二、真题讲解

**例1 (2022国考改编)**：
某市2020年高新技术产业产值为480亿元，计划2021年增长10%，但由于技术革新，2021年实际产值比计划增长了10%。问2022年为实现产值达到635.25亿元的目标，需要比2021年增长百分之多少？

A. 12.5% <br />
B. 15% <br />
C. 17.5% <br />
D. 20% <br />

<BlurredAnswer>
**解题思路：**

本题属于增长率问题，核心是理解“比计划增长了10%”的含义，并进行精确计算。

1.  <span style={{color:'green'}}>第一步：计算2021年的计划产值。</span>
    根据题意，2020年产值为480亿元，计划2021年增长10%。
    $$ \text{2021年计划产值} = 480 \times (1 + 10\%) = 480 \times 1.1 = 528 \text{ 亿元} $$ 

2.  <span style={{color:'green'}}>第二步：计算2021年的实际产值。</span>
    2021年实际产值比<span style={{color:'red'}}>计划产值</span>增长了10%。
    $$ \text{2021年实际产值} = 528 \times (1 + 10\%) = 528 \times 1.1 = 580.8 \text{ 亿元} $$ 
    <span style={{color:'orange'}}>易错点：这里的增长基础是“2021年的计划产值（528亿元）”，而不是“2020年的产值（480亿元）”。因此2021年相对于2020年的总增长率是 $1.1 \times 1.1 - 1 = 1.21 - 1 = 21\%$，而不是 $10\% + 10\% = 20\%$。</span>

3.  <span style={{color:'green'}}>第三步：计算2022年的目标增长率。</span>
    2022年的目标产值是635.25亿元，我们需要计算相比于2021年实际产值需要增长多少。
    $$ \text{增长率} = \frac{\text{增长量}}{\text{基期量}} = \frac{\text{目标产值} - \text{实际产值}}{\text{实际产值}} $$ 
    $$ \text{增长率} = \frac{635.25 - 580.8}{580.8} = \frac{54.45}{580.8} $$ 

4.  <span style={{color:'purple'}}>第四步：速算技巧-直除法。</span>
    使用直除法计算 $54.45 \div 580.8$，这是资料分析中更高效的计算技巧。
    
    **步骤1：调整小数点**
    将被除数和除数同时乘以100，转化为整数运算：
    $$ 54.45 \div 580.8 = \frac{5445}{58080} $$
    
    **步骤2：直除计算**
    $$ \frac{5445}{58080} \approx 0.09375 $$
    
    计算过程：
    - 5445 ÷ 58080 = 0.09375（直接计算）
    - 或者：54.45 ÷ 580.8 ≈ 0.09375
    
    **步骤3：转化为百分数**
    $$ 0.09375 = 9.375\% $$ 

**答案：C**

**技巧总结：**
在资料分析题中，当遇到复杂的除法时，直除法往往比分数约分更加高效直接。对于像 $54.45/580.8$ 这样的计算，可以直接进行除法运算，或者先调整小数点位置转化为整数运算，避免繁琐的约分过程。直除法特别适用于选项差距较大的题目，能够快速得到近似结果。

**例2**：
计算 $96 \times 1.1$

A. 105.6 <br />
B. 106.6 <br />
C. 107.6 <br />
D. 108.6 <br />

**解题思路：**

1.  <span style={{color:'green'}}>第一步：理解乘以1.1的含义。</span>
    根据乘法分配律：
    $$96 \times 1.1 = 96 \times (1 + 0.1) = 96 + 96 \times 0.1$$

2.  <span style={{color:'green'}}>第二步：计算96的十分之一。</span>
    $96$ 的十分之一是 $9.6$（小数点向左移动一位）。

3.  <span style={{color:'green'}}>第三步：将结果相加。</span>
    $96 + 9.6 = 105.6$

因此，答案为 A. 105.6

**例4**：
计算 $85 \times 0.9$

A. 74.5 <br />
B. 75.5 <br />
C. 76.5 <br />
D. 77.5 <br />

<BlurredAnswer>
**解题思路：**

1.  <span style={{color:'green'}}>第一步：理解乘以0.9的含义。</span>
    根据乘法分配律：
    $$85 \times 0.9 = 85 \times (1 - 0.1) = 85 - 85 \times 0.1$$

2.  <span style={{color:'green'}}>第二步：计算85的十分之一。</span>
    $85$ 的十分之一是 $8.5$（小数点向左移动一位）。

3.  <span style={{color:'green'}}>第三步：用原数减去该结果。</span>
    $85 - 8.5 = 76.5$

因此，答案为 C. 76.5

## 三、技巧总结

1.  **核心公式**：
    *   $A \times 1.1 = A + A/10$ (错位加)
    *   $A \times 1.5 = A + A/2$ (加一半)
    *   $A \times 0.9 = A - A/10$ (错位减)

2.  **技巧拓展**：
    *   $A \times 1.2 = A + A/5$
    *   $A \times 0.8 = A - A/5$
    *   $A \times 2.5 = A \times 10 / 4 = 2A + A/2$
    *   $A \div 1.25 = A \div (5/4) = A \times 4/5 = A \times 0.8$

3.  **注意事项**：
    *   <span style={{color:'orange'}}>看清增长基数</span>：在连续增长或减少的问题中，要明确每一次变化的基准量是谁。
    *   <span style={{color:'orange'}}>区分“提高到”和“提高了”</span>：“提高到原来的1.5倍”指 $A \times 1.5$；“提高了原来的1.5倍”指 $A \times (1+1.5) = A \times 2.5$。
    *   <span style={{color:'orange'}}>灵活运用</span>：当遇到复杂计算时，优先考虑是否可以转化为特殊数字乘法，或者使用代入排除、估算法等其他技巧。
