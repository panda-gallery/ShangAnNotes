# 分数比较-差分法

## 核心概念

### 1. 差分法的本质理解

想象一下，你和朋友比较两家奶茶店的性价比：
- A店：30元买5杯奶茶，性价比 = $\frac{5}{30} = \frac{1}{6}$
- B店：25元买4杯奶茶，性价比 = $\frac{4}{25}$

哪家更划算呢？直接计算小数比较麻烦，这时候<span style={{color:'red'}}>差分法</span>就派上用场了！

### 2. 差分法公式推导

设有两个分数：$\frac{a}{b}$ 和 $\frac{c}{d}$，其中 $a > c$，$b > d$（分子分母同大同小）

我们要比较它们的大小，等价于判断：
$$\frac{a}{b} - \frac{c}{d}$$ 
的正负性。

通分得：
$$\frac{a}{b} - \frac{c}{d} = \frac{ad - bc}{bd}$$

由于 $b > 0$，$d > 0$，所以 $bd > 0$，因此：
- 若 $ad - bc > 0$，则 $\frac{a}{b} > \frac{c}{d}$
- 若 $ad - bc < 0$，则 $\frac{a}{b} < \frac{c}{d}$
- 若 $ad - bc = 0$，则 $\frac{a}{b} = \frac{c}{d}$

现在关键是如何快速判断 $ad - bc$ 的正负性。

<span style={{color:'purple'}}>巧妙之处</span>：我们构造一个新的分数 $\frac{a-c}{b-d}$（差分数），然后：

$$ad - bc = (a-c)d + cd - bc = (a-c)d - c(b-d) = d(a-c) - c(b-d)$$

因此：
$$\frac{ad - bc}{bd} = \frac{d(a-c) - c(b-d)}{bd} = \frac{a-c}{b} - \frac{c}{d}$$

这样我们就得到了<span style={{color:'red'}}>差分法的核心公式</span>：

$$\frac{a}{b} \text{ 与 } \frac{c}{d} \text{ 的大小关系} \Leftrightarrow \frac{a-c}{b-d} \text{ 与 } \frac{c}{d} \text{ 的大小关系}$$

### 4. 差分法比较方法详解

<span style={{color:'green'}}>具体比较步骤</span>：

**第一步：确定大分数和小分数**
- 设 $\frac{a}{b}$ 为分子较大的分数（大分数）
- 设 $\frac{c}{d}$ 为分子较小的分数（小分数）
- 确保 $a > c$ 且 $b > d$

**第二步：计算差分数**
$$\text{差分数} = \frac{a-c}{b-d}$$

**第三步：比较差分数与小分数**
- 计算小分数 $\frac{c}{d}$ 的值（可以是近似值）
- 比较差分数与小分数的大小关系

**第四步：根据比较结果得出结论**

### 5. 结果解释的数学原理

为什么这个规则成立？让我们从数学角度理解：

由前面的推导我们知道：
$$\frac{a}{b} - \frac{c}{d} = \frac{a-c}{b} - \frac{c}{d}$$

<span style={{color:'purple'}}>关键洞察</span>：上式可以重写为：
$$\frac{a}{b} - \frac{c}{d} = \frac{a-c}{b-d} \cdot \frac{b-d}{b} - \frac{c}{d}$$

由于 $b > d > 0$，所以 $\frac{b-d}{b} < 1$，但这个系数不影响大小关系的判断。

更直观的理解：
- 差分数 $\frac{a-c}{b-d}$ 反映了"增量比率"
- 小分数 $\frac{c}{d}$ 是"基准比率"
- 当增量比率 > 基准比率时，说明大分数的增长"超过了预期"，因此大分数更大
- 当增量比率 < 基准比率时，说明大分数的增长"不及预期"，因此小分数更大

### 6. 实际应用中的技巧

<span style={{color:'orange'}}>计算技巧</span>：
1. **差分数优先化简**：如 $\frac{15}{3} = 5$，$\frac{24}{8} = 3$
2. **小分数估算即可**：通常保留2位小数足够判断
3. **明显大小关系时无需精确计算**：如差分数是8，小分数约为2，明显8>2
4. **利用分数性质**：如 $\frac{100}{20} = 5$，$\frac{99}{19} \approx 5.2$，可快速判断

### 7. 差分法使用条件

<span style={{color:'orange'}}>重要限制</span>：
1. **分子分母必须同向变化**：即 $a > c$ 且 $b > d$，或者 $a < c$ 且 $b < d$
2. **差分数的分母不能为0**：即 $b \neq d$
3. **适用于接近的分数比较**：差分法在分数值接近时效果最佳

## 真题讲解

### 主题一：基础分数比较

**例1**（模拟国考题型）：
比较 $\frac{127}{23}$ 和 $\frac{119}{22}$ 的大小。

A. $\frac{127}{23} > \frac{119}{22}$ <br />
B. $\frac{127}{23} < \frac{119}{22}$ <br />
C. $\frac{127}{23} = \frac{119}{22}$ <br />
D. 无法确定 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：检查适用条件**
- 分子：127 > 119 ✓
- 分母：23 > 22 ✓
- 满足"分子分母同大同小"的条件

**第二步：确定大分数和小分数**
- 大分数：$\frac{127}{23}$
- 小分数：$\frac{119}{22}$

**第三步：计算差分数**
$$\text{差分数} = \frac{127-119}{23-22} = \frac{8}{1} = 8$$

**第四步：比较差分数与小分数**
- 差分数：8
- 小分数：$\frac{119}{22} \approx 5.41$
- 因为 $8 > 5.41$

**第五步：得出结论**
根据差分法规则：$\frac{8}{1} > \frac{119}{22} \Rightarrow \frac{127}{23} > \frac{119}{22}$

**答案：A**
</BlurredAnswer>

### 主题二：小数分数混合比较

**例2**（模拟省考题型）：
比较 $\frac{456}{73.2}$ 和 $\frac{441}{71.8}$ 的大小。

A. 前者大 <br />
B. 后者大 <br />
C. 相等 <br />
D. 无法比较 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：检查适用条件**
- 分子：456 > 441 ✓
- 分母：73.2 > 71.8 ✓
- 满足使用条件

**第二步：计算差分数**
$$\text{差分数} = \frac{456-441}{73.2-71.8} = \frac{15}{1.4}$$

**第三步：化简差分数**
$$\frac{15}{1.4} = \frac{150}{14} = \frac{75}{7} \approx 10.71$$

**第四步：计算小分数的近似值**
$$\frac{441}{71.8} \approx 6.14$$

**第五步：比较并得出结论**
因为 $10.71 > 6.14$，所以 $\frac{456}{73.2} > \frac{441}{71.8}$

**答案：A**
</BlurredAnswer>

### 主题三：增长率比较问题

**例3**（模拟联考题型）：
某地区2023年GDP为2840亿元，2024年为2976亿元；另一地区2023年GDP为1960亿元，2024年为2058亿元。比较两地区的GDP增长率。

A. 第一地区增长率高 <br />
B. 第二地区增长率高 <br />
C. 增长率相等 <br />
D. 无法确定 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：理解题意**
- 第一地区增长率：$\frac{2976-2840}{2840} = \frac{136}{2840}$
- 第二地区增长率：$\frac{2058-1960}{1960} = \frac{98}{1960}$

**第二步：检查适用条件**
- 分子：136 > 98 ✓
- 分母：2840 > 1960 ✓
- 可以使用差分法

**第三步：计算差分数**
$$\text{差分数} = \frac{136-98}{2840-1960} = \frac{38}{880} = \frac{19}{440}$$

**第四步：比较差分数与小分数**
- 差分数：$\frac{19}{440} \approx 0.0432$
- 小分数：$\frac{98}{1960} = \frac{49}{980} \approx 0.0500$

**第五步：得出结论**
因为 $0.0432 < 0.0500$，所以第一地区增长率 < 第二地区增长率

**答案：B**
</BlurredAnswer>

### 主题四：效率比较问题

**例4**（模拟国考题型）：
甲工厂用45天生产了180件产品，乙工厂用38天生产了156件产品。比较两工厂的生产效率。

A. 甲工厂效率高 <br />
B. 乙工厂效率高 <br />
C. 效率相等 <br />
D. 无法比较 <br />

<BlurredAnswer>
**解题步骤：**

**第一步：建立效率模型**
- 甲工厂效率：$\frac{180}{45} = 4$ 件/天
- 乙工厂效率：$\frac{156}{38}$ 件/天

**第二步：检查适用条件**
- 分子：180 > 156 ✓
- 分母：45 > 38 ✓
- 满足使用条件

**第三步：计算差分数**
$$\text{差分数} = \frac{180-156}{45-38} = \frac{24}{7} \approx 3.43$$

**第四步：计算小分数**
$$\frac{156}{38} \approx 4.11$$

**第五步：比较并得出结论**
因为 $3.43 < 4.11$，所以 $\frac{180}{45} < \frac{156}{38}$

即甲工厂效率 < 乙工厂效率

**答案：B**
</BlurredAnswer>

### 技巧二：特殊情况的快速判断

<span style={{color:'red'}}>当差分数等于1时</span>，有特殊性质：

若 $\frac{a-c}{b-d} = 1$，即 $a-c = b-d$，则：
$$\frac{a}{b} - \frac{c}{d} = \frac{ad-bc}{bd} = \frac{(c+k)d - c(d+k)}{b \cdot d} = \frac{cd + kd - cd - ck}{bd} = \frac{k(d-c)}{bd}$$

其中 $k = a-c = b-d$

## 技巧总结

### 1. 使用步骤口诀

<span style={{color:'purple'}}>"同大同小差分算，差分小数比一比"</span>

1. **同大同小**：检查分子分母是否同向变化
2. **差分算**：计算差分数 $\frac{a-c}{b-d}$
3. **比一比**：比较差分数与小分数的大小

### 2. 常见错误及避免方法

<span style={{color:'orange'}}>易错点1</span>：忘记检查使用条件
- **避免方法**：每次使用前必须确认分子分母同向变化

<span style={{color:'orange'}}>易错点2</span>：搞混大分数和小分数
- **避免方法**：始终以分子较大的为"大分数"

<span style={{color:'orange'}}>易错点3</span>：差分数计算错误
- **避免方法**：仔细计算 $(a-c)$ 和 $(b-d)$，注意符号

### 3. 适用题型总结

| 题型 | 特征 | 示例 |
|------|------|------|
| 基础分数比较 | 纯分数形式 | $\frac{127}{23}$ vs $\frac{119}{22}$ |
| 增长率比较 | 增长量/基期值 | GDP、人口增长率 |
| 效率比较 | 产量/时间 | 工作效率、生产效率 |
| 密度比较 | 质量/体积 | 人口密度、物质密度 |
| 平均数比较 | 总量/数量 | 平均分、平均收入 |

### 4. 提升速度的建议

1. **熟练掌握基本运算**：加减乘除要快速准确
2. **培养数感**：对常见分数的小数值要有直觉
3. **多练习**：通过大量练习形成条件反射
4. **灵活运用**：<span style={{color:'green'}}>当差分数明显大于或小于小分数时，可以不用精确计算</span>

### 5. 与其他方法的比较

| 方法 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|

| 通分法 | 适用性广 | 计算复杂 | 所有分数比较 |
| 化小数法 | 直观易懂 | 计算量大 | 简单分数 |
| 交叉相乘法 | 准确可靠 | 数字较大时计算复杂 | 精确比较 |
