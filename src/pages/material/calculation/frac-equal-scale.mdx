# 除法-等比缩放法

## 核心概念

等比缩放法（也叫凑同比）是资料分析中一种重要的速算技巧，其<span style={{color:'red'}}>核心原理是通过同比例调整分子分母，使计算变得更加简便</span>。

### 基本原理推导

让我们用一个生活中的例子来理解这个概念：

假设小明去超市买苹果，苹果的价格是每斤8元，他买了5斤，总共花费40元。如果苹果涨价到每斤10元，他要花同样的40元，能买多少斤呢？

我们可以列出比例：$\frac{8}{5} = \frac{10}{x}$

解得：$x = \frac{5 \times 10}{8} = \frac{50}{8} = 6.25$斤

但是，我们也可以这样思考：如果价格从8元变成10元，增加了2元，那么相应地，重量也要按比例调整。

<span style={{color:'red'}}>关键发现</span>：$\frac{8}{5} = \frac{8+2}{5+1.25} = \frac{10}{6.25}$

这就是等比缩放的基本思想：<span style={{color:'purple'}}>当分子分母按相同比例变化时，分数值保持不变</span>。

### 数学表达式

$$\frac{a}{b} = \frac{a \pm ka}{b \pm kb} = \frac{a(1 \pm k)}{b(1 \pm k)}$$

其中k为缩放系数。

### 实际应用中的变形

在资料分析中，我们通常采用以下策略：

1. **凑整数**：如 $193 \rightarrow 200$，$424 \rightarrow 400$，$1.18 \rightarrow 1.2$
2. **凑百化分数**：如 $1.23 \rightarrow 1.25$，$1.62 \rightarrow 1.67$，$30 \rightarrow 33.3$
3. **凑便于计算的数**：如 $121 \rightarrow 125$（便于×8）

## 操作步骤详解

### 第一步：数据预处理
分子分母通常取<span style={{color:'red'}}>三位有效数字</span>（四舍五入）
- 分子分母只有两位时也可以用，一个三位一个两位，两位后添0
- <span style={{color:'orange'}}>分子分母尽量保持位数一致</span>

### 第二步：确定调整策略
将分母凑成：
- 整十、整百、整千（10、100、1000……）
- 方便约分的数字（125、167等）
- 便于百化分的数字

### 第三步：等比例缩放
根据分母的调整幅度，确定分子的调整方式：
- **加减一位数**：按首位倍数调整
- **加减两位数**：按首两位倍数调整

## 真题讲解

### 主题一：基础除法计算

**例1**（2024国考模拟）：
计算 $\frac{1284}{104}$ 的值，最接近：

A. 11.9 <br/>
B. 12.3 <br/>
C. 12.7 <br/>
D. 13.2 <br/>

<BlurredAnswer>
**方法1：直除法硬算**

直接计算：$1284 \div 104$

使用长除法：$1284 \div 104 = 12.346...$

最接近的答案是B. 12.3

**方法2：倍数关系巧算**

第一步：观察倍数关系
- 1284是104的大概12倍（$104 \times 12 = 1248$）

第二步：利用等比缩放
- 分母：$104 - 4 = 100$
- 分子：$1284 - 12 \times 4 = 1284 - 48 = 1236$

第三步：计算结果
$$\frac{1236}{100} = 12.36$$

最接近的答案是B. 12.3

**验证**：用计算器 $1284 \div 104 = 12.346...$，确实最接近12.3
</BlurredAnswer>


### 主题二：增长量计算中的应用

**例3**（2021福建省考）：
某地区2020年GDP为4901万元，同比增长13.6%，求2020年的增长量：

$$\frac{4901}{1 + 13.6\%} \times 13.6\% = ?$$

A. 586.7 <br/>
B. 579.5 <br/>
C. 568.2 <br/>
D. 559.4 <br/>

<BlurredAnswer>
**解题步骤：**

第一步：所有数保留三位，如下：
$$\frac{490}{114} \times 136$$

第二步：约分法整。此时114要变成136，114+22=136。

第三步：计算分数值。
$$\frac{490}{114} = 4.3$$

第四步：计算结果
$$4.3 \times 22 + 490 = 94.6 + 490 = 584.6$$

最接近的选项为A. 586.7，所以选择A。
</BlurredAnswer>

### 主题三：复杂分数的处理

**例4**（2024模拟题）：
计算 $\frac{577}{135}$ 的值：

A. 4.1 <br/>
B. 4.3 <br/>
C. 4.5 <br/>
D. 4.7 <br/>

<BlurredAnswer>
**解题思路：**

第一步：分解分子
- 将577分解为：577 = 135 × 4 + 37
- 即：$$\frac{577}{135} = \frac{135 \times 4 + 37}{135} = 4 + \frac{37}{135}$$

第二步：估算小数部分
- <span style={{color:'red'}}>关键技巧</span>：比较 $\frac{37}{135}$ 与各选项的小数部分
- $\frac{37}{135} ≈ \frac{37}{135} ≈ 0.274$
- 因为 $1.37 \times 2 = 2.74 < 3.7 < 1.37 \times 3 = 4.11$
- 所以 $\frac{37}{135}$ 更接近 0.3

第三步：确定答案
- $4 + \frac{37}{135} ≈ 4 + 0.3 = 4.3$

答案选择B. 4.3

**验证**：577÷135 = 4.274，最接近4.3
</BlurredAnswer>

### 主题四：百分数转换中的应用

**例5**（2023深圳市考）：
某企业2022年营业收入1214.53万元，同比增长14.8%，求增长量：

$$\frac{1214.53}{1 + 14.8\%} \times 14.8\% = ?$$

A. 149.37 <br/>
B. 151.76 <br/>
C. 156.57 <br/>
D. 161.25 <br/>

<BlurredAnswer>
**解题步骤：**

第一步：数据简化
$$\frac{121}{115} \times 148$$

第二步：分母调整
- 115 + 33 = 148（加两位数33）

第三步：同比例缩放
- 33 = 11×3（首两位倍数）
- 分子调整：121 + 12×3 = 121 + 36 = 157

第四步：计算
$$\frac{157}{148} \times 148 = 157$$

答案选择C. 156.57
</BlurredAnswer>

### 主题五：比值比较中的应用

**例6**（2024国考模拟）：
比较 $\frac{2847}{96}$ 和 $\frac{3156}{108}$ 的大小：

A. 前者大 <br/>
B. 后者大 <br/>
C. 相等 <br/>
D. 无法确定 <br/>

<BlurredAnswer>
**解题思路：**

**方法：分母差值法**

1. 计算分母差值：108 - 96 = 12

2. 计算分母变化率：$\frac{12}{96} = \frac{1}{8}$

3. 计算分子差值：3156 - 2847 = 309

4. 判断分子变化率：$\frac{309}{2847} = \frac{309}{2847}$

5. 关键比较：$\frac{2847}{8} = 355.875$
   
   而分子差值 309 < 355.875
   
   即：$\frac{309}{2847} < \frac{1}{8}$

6. <span style={{color:'red'}}>当分子的变化率小于分母的变化率时，分数值变小</span>

因此：$\frac{2847}{96} > \frac{3156}{108}$

答案：A. 前者大
</BlurredAnswer>
