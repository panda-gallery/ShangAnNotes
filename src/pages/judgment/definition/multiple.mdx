# 多定义判断

在公务员考试的判断推理模块中，多定义判断是一种常见的题型。它通常在一个题干中给出两个或多个相关的定义，然后要求考生根据这些定义，对选项中的情景进行判断、区分或匹配。

这种题型考察的是考生快速阅读、准确理解、信息筛选和对比分析的能力。很多同学看到长篇的定义就心生畏惧，但其实只要我们掌握了核心方法，就能化繁为简，精准破题。

## 核心概念

我们可以把解多定义题想象成一个“按图索骥”的游戏。你手里有一份“藏宝图”（题干里的多个定义），需要根据图上的线索，从几个看似相似的“地点”（选项）中，找到真正的“宝藏”（正确答案）。关键在于，你要能看懂图上每一条线索的精确含义，并且知道这些线索之间的关系。

在考试中，多定义的关系主要有以下三种：

### 1. 包含关系

**核心特征**：一个定义是另一个定义下的子类或特殊情况，我们称之为“母子关系”。子定义必须<span style={{color:'red'}}>首先满足母定义的所有条件</span>，然后再满足自己特有的条件。

**举个例子**：
我们来定义两个概念：
- **母定义（交通工具）**：一种用于载人或载物的装置。
- **子定义（汽车）**：一种<span style={{color:'orange'}}>拥有四个轮子和自带动力系统</span>的**交通工具**。

这里的“汽车”就是“交通工具”的子定义。一个东西要被判断为“汽车”，它必须首先满足“交通工具”的定义（能载人或载物），其次还要满足它自己的特殊条件（四个轮子、自带动力）。一辆自行车，虽然是交通工具，但因为它不满足“自带动力系统”，所以它不是汽车。

**解题关键**：判断一个选项是否符合子定义时，要先用母定义的条件去“筛”一遍。

### 2. 并列关系

**核心特征**：两个或多个定义处于同一层级，它们共同描述一个大类下的不同小类，彼此独立但又有联系。

**举个例子**：
我们来定义两种常见的支付方式：
- **定义一（扫码支付）**：用户通过手机等设备扫描二维码，从而完成资金转移的支付方式。
- **定义二（刷脸支付）**：用户通过面部识别技术，验证身份后完成资金转移的支付方式。

“扫码支付”和“刷脸支付”都是“电子支付”的并列分支。它们的目的是一样的（完成支付），但实现的核心技术和方式是截然不同的。一个是“扫码”，一个是“刷脸”。

**解题关键**：<span style={{color:'red'}}>精准识别并比较各个定义之间的“核心差异点”</span>。这是解题的钥匙。

### 3. 匹配关系

**核心特征**：题干给出多个定义，并附带多个案例，要求将定义与案例进行一一对应。

**举个例子**：
这就像是医院的分诊台。
- **定义一（内科）**：处理心、肺、胃等内部器官疾病的科室。
- **定义二（外科）**：通过手术等方式处理外伤、肿瘤等疾病的科室。
- **定义三（五官科）**：处理眼、耳、鼻、喉相关疾病的科室。

现在来了三个病人（选项）：
- **案例A**：张三感觉心脏不舒服。
- **案例B**：李四打球时手臂骨折了。
- **案例C**：王五最近总觉得耳朵疼。

你的任务就是把这三位病人精确地匹配到对应的科室去。

**解题关键**：<span style={{color:'red'}}>从案例（选项）入手，分析其特征</span>，然后“倒回去”找最符合的定义进行匹配，这样效率最高。

## 真题讲解

接下来，我们通过几道经典的真题，来实际应用一下上述的概念和方法。

### 主题：包含关系 - 属种与反变

**例1**：（2015国考）概念的内涵是指概念所反映的事物具有的本质属性（或特有属性），它是从质的方面反映概念的；概念的外延是指具有概念所反映的本质属性（或特有属性）的那些事物，它是从量的方面反映概念的。内涵与外延的反变关系是指具有属种关系的概念之间所具有的一种相互联系、相互制约的关系，一个概念的外延越大，它的内涵越少；一个概念外延越小，它的内涵越多。反之，一个概念的内涵越多，它的外延越小；一个概念内涵越少，它的外延越大。

根据上述定义，下列哪组概念体现了概念内涵与外延的反变关系？

- A. 司法机关，最高审判机关，最高人民法院
- B. 水杯，热水杯，大号热水杯
- C. 中国，新疆，乌鲁木齐
- D. 中国人，文学家，鲁迅

<BlurredAnswer>
### 思路剖析

**第一步：拆解定义要件。**
本题的核心是“内涵与外延的反变关系”。要满足这个关系，必须同时符合两个核心要件：
1.  <span style={{color:'red'}}>要件一（前提）：概念之间必须是“属种关系”</span>。属种关系，即“大类”与“小类”的关系（如“动物”与“狗”）。
2.  <span style={{color:'red'}}>要件二（规律）：内涵与外延要“反向变化”</span>。即范围（外延）越大的概念，其包含的属性（内涵）越少；反之，范围越小，属性越多。

**第二步：逐项分析选项，判断是否同时满足两个要件。**

-   **A选项**：“司法机关”是一个大类，“最高审判机关”是司法机关中的一种，而“最高人民法院”是最高审判机关的具体指代。这组概念层层包含，范围逐渐变小（外延变小），而其限定条件越来越多（内涵增多）。例如，“最高审判机关”比“司法机关”多了“最高审判”的限定。这符合属种关系和反变规律。

-   **B选项**：“水杯”是大类，“热水杯”是水杯中的一种（增加了“装热水”的属性），“大号热水杯”又是热水杯中的一种（增加了“大号”的属性）。这同样是清晰的属种关系，外延依次减小，内涵依次增多。符合定义。

-   **C选项**：“中国”、“新疆”、“乌鲁木齐”之间是<span style={{color:'orange'}}>整体与部分的关系</span>，而非属种关系。新疆是中国的“一部分”，而不是“一种中国”。这是最常见的易错点，必须区分“is a part of”和“is a kind of”。因此，不满足要件一。

-   **D选项**：“中国人”和“文学家”是属种关系，但“鲁迅”是一个<span style={{color:'orange'}}>具体的个体（实例）</span>，而不是一个“概念”或“类别”。定义是用来描述概念之间关系的，所以此项不符合。

**第三步：比较A和B，确定最优答案。**
题目要求选出“体现”了这种关系的选项。A和B都体现了这种关系。但在公务员考试单选题中，通常会有一个最佳选项或存在一个不易察셔的陷阱。然而，此题在原始考题中，A、B、C均为迷惑性很强的选项，D是明显错误。A和B的逻辑结构是最严谨的属种关系。通常在这种情况下，会选择最典型、最无争议的那个。B选项的“水杯 -> 热水杯 -> 大号热水杯”在日常逻辑中层层递进的关系非常清晰。

（注：此题在不同解析版本中答案可能存在争议，但其考察的核心在于区分属种关系、整体部分关系以及类与个体的关系。在此我们以B为典型范例进行讲解）

**因此，选择 B。**
</BlurredAnswer>

### 主题：包含关系 - 母子定义的嵌套

**例2**：（2023国考）职业锚是指个体在职业生涯中长期稳定地持有的一种价值观、动机和需求组合，是个体在面临职业选择时，无论如何都不会放弃的职业核心。
根据职业锚的理论，可以将个体的职业锚分为不同类型，其中，技术/职能型职业锚是指个体围绕特定的技术或职能领域发展其职业生涯，追求在该领域不断提升专业能力，并以此为基础获得职业上的成功和满足感。

根据上述定义，下列哪项最能体现技术/职能型职业锚？

- A. 小张原本是一名优秀的计算机教师，但他对园艺更有兴趣，最终辞职开了一家花店
- B. 李主任曾是医院里技术精湛的外科医生，后来转任分管行政的副院长
- C. 一位著名演员在事业高峰期选择暂时息影，去商学院学习工商管理课程
- D. 老王年轻时从佛跳墙学徒做起，凭借不懈努力，最终成为五星级酒店的行政主厨

<BlurredAnswer>
### 思路剖析

**第一步：识别母子定义，并拆解要件。**
-   **母定义（职业锚）**：核心是<span style={{color:'red'}}>“无论如何都不会放弃的职业核心”</span>，强调“长期稳定”。
-   **子定义（技术/职能型职业锚）**：它首先得是一个“职业锚”，其次，这个锚的核心必须是：
    1.  **要件一**：围绕<span style={{color:'red'}}>“特定的技术或职能领域”</span>。
    2.  **要件二**：追求在该领域<span style={{color:'red'}}>“不断提升专业能力”</span>。

**第二步：分析选项是否同时满足子定义的两个要件。**

-   **A选项**：小张从“计算机”转行到“园艺”，完全改变了技术领域。这直接违背了“职业锚”的“稳定”和“不放弃”的核心，更不符合“技术/职能型”的“围绕特定技术”的要求。

-   **B选项**：李主任从“外科医生”（技术岗）转为“行政副院长”（管理岗）。虽然仍旧在医院，但其核心职能从“技术”转向了“行政管理”。这违背了“围绕特定技术领域发展”的要求。

-   **C选项**：演员去学习工商管理，是从“表演艺术”这个职能领域，准备转向“商业管理”领域。这同样是职业路径的转变，而非在原有技术上的深化。

-   **D选项**：老王的职业生涯始终围绕<span style={{color:'red'}}>“烹饪”</span>这一特定技术领域。他从“学徒”做到“行政主厨”，是典型地在同一技术领域内<span style={{color:'red'}}>“不断提升专业能力”</span>并获得成功的路径。这完美地契合了“技术/职能型职业锚”的所有要件。

**因此，选择 D。**
</BlurredAnswer>

### 主题：并列关系 - 寻找核心差异

**例3**：（2021国考）社会类别化是指个体基于性别、年龄、种族、国别等方面的相似性，将他人划归到特定社会群体中的过程。
自我类别化是个体将自己视为某一社会群体的成员，并体现出该群体成员共有的态度、情感和行为的过程。

根据上述定义，下列哪项属于自我类别化？

- A. 王女士在异国他乡的聚会上，主动用方言和几位同乡交流，感觉格外亲切
- B. 父母从小就觉得小明性格沉稳，适合做一名严谨的公务员
- C. 同学们因为小李做事认真负责，一致推举他担任班级的卫生委员
- D. 某公司在招聘时，倾向于选择名校毕业的求职者

<BlurredAnswer>
### 思路剖析

**第一步：对比并列定义，找到核心差异点。**
这两个定义是并列关系，都是“类别化”，但方向完全相反。
-   **社会类别化**：核心是<span style={{color:'red'}}>“将他人划归...”</span>，行为的主体是“我/我们”，对象是“他/他们”。简单说就是“给别人分类贴标签”。
-   **自我类别化**：核心是<span style={{color:'red'}}>“将自己视为...”</span>，行为的主体和对象都是“我/自己”。简单说就是“自己认同并归入某个群体”。
-   **核心差异**：<span style={{color:'red'}}>行为发出的对象不同</span>，一个是向外（对他人），一个是向内（对自己）。

**第二步：分析选项的行为方向，进行匹配。**
题目问的是“自我类别化”，所以我们要找一个“将自己归入群体”的例子。

-   **A选项**：王女士在特定场合（异国他乡），通过行为（说方言）将自己归入“同乡”这个群体，并产生了相应的情感（感觉亲切）。行为的主体和对象都是王女士自己，完美符合“自我类别化”。

-   **B选项**：“父母觉得小明...”，行为主体是“父母”，对象是“小明”。这是父母在给小明进行类别划分。属于社会类别化。

-   **C选项**： “同学们推举他...”，行为主体是“同学们”，对象是“小李”。这是同学们在给小李进行类别划分。属于社会类别化。

-   **D选项**： “公司倾向于选择...”，行为主体是“公司”，对象是“求职者”。这是公司在给求职者进行类别划分。属于社会类别化。

**因此，选择 A。**
</BlurredAnswer>

### 主题：并列关系 - 法律概念区分

**例4**：（2024浙江）自然法主张一定的权利因为人类本性中的美德而固然存在，由自然赋予；成文法是指国家机关依照一定的程序制定和颁布的，表现为条文形式的法律文件；习惯法是指独立于国家制定法之外，依据某种社会权威和社会组织，具有一定强制性的行为规范的总和。

根据上述定义，下列属于习惯法的是：

- A. 举事以为人者，众助之；举事以自为者，众去之
- B. 凡教唆词讼，及为人作词状，增减情罪诬告人者，与犯人同罪
- C. 上圣卓然先行敬让博爱之德者，众心说而从之。从之成群，是为君矣；归而往之，是为王矣
- D. 议得我辈同行业者，不与现在行家弟兄亲戚，以及管账之人合伙合业，以防明充暗顶之渐，违者公罚

<BlurredAnswer>
### 思路剖析

**第一步：拆解定义要件，聚焦核心。**
题目要求判断“习惯法”，我们直接看“习惯法”的定义。
-   **习惯法**：
    1.  **要件一**：<span style={{color:'red'}}>独立于国家制定法之外</span>。
    2.  **要件二**：依据<span style={{color:'red'}}>某种社会权威和社会组织</span>（如行会、宗族）。
    3.  **要件三**：具有<span style={{color:'red'}}>一定强制性</span>的行为规范（有惩罚措施）。

同时，为了区分，我们也要了解其他两个定义的核心：
-   **自然法**：核心是“人类本性”、“美德”、“自然赋予”，偏向于道德和哲学层面。
-   **成文法**：核心是“国家机关”、“制定和颁布”，是正式的法律文件。

**第二步：逐项分析，进行匹配。**

-   **A选项**：出自《淮南子·兵略训》，意思是为众人做事，大家就帮助他；为自己做事，大家就离弃他。这是一种道德倡导，不具备强制性，更接近自然法的理念。不符合习惯法。

-   **B选项**：出自《大清律例》，是清朝的国家法律。这明确属于“国家机关”颁布的“成文法”。不符合习惯法。

-   **C选项**：出自《汉书·刑法志》，描述的是圣人以德服人，人们自愿追随形成君王。这同样是一种关于统治合法性的道德和哲学论述，不具备强制性规范。不符合习惯法。

-   **D选项**：出自《吴县纱缎业行规条约碑》。
    -   “我辈同行业者”的“议得”，说明这是<span style={{color:'red'}}>行业内部（社会组织）</span>的规定，而非国家制定，满足要件一和二。
    -   “不与……合伙合业”，“违者公罚”，明确规定了行为规范和<span style={{color:'red'}}>惩罚措施（强制性）</span>，满足要件三。
    -   完美匹配“习惯法”的所有定义。

**因此，选择 D。**
</BlurredAnswer>

### 主题：并列关系 - 多条件匹配

**例5**：（2024国考）中医七方指的是大方、小方、缓方、急方、奇方、偶方、复方七种方剂的总称。其中缓方是指药性缓和，治疗病势缓慢需长期服用的方剂；急方是指药性峻猛，治疗病势急重急于取效的方剂；奇方是指单数药物组成的方剂；偶方是指双数药物组成的方剂。

根据上述定义，下列既属于缓方又属于偶方的是：

- A. 参附汤：人参15克、附子30克，有回阳、益气、固脱之功，常用于元气大亏、阳气暴脱之症
- B. 甘草汤：甘草6克，主治少阴咽痛，兼治舌肿，有清热解毒之功效
- C. 四君子汤：人参、甘草、茯苓、白术各等分，主治脾胃气虚证，症见面色萎黄、语声低微、气短乏力等
- D. 生脉散：人参9克、麦门冬9克、五味子6克，为补益剂，具有益气生津、敛阴止汗之功效

<BlurredAnswer>
### 思路剖析

**第一步：拆解要件，锁定目标。**
题目要求同时满足两个条件：
1.  **缓方**：<span style={{color:'red'}}>药性缓和</span>，治疗<span style={{color:'red'}}>病势缓慢</span>。
2.  **偶方**：由<span style={{color:'red'}}>双数</span>药物组成。

**第二步：逐项筛选，双重验证。**
我们用这两个“筛子”去过滤选项。

-   **A选项**：
    -   **药物数量**：人参、附子，共2味药，是双数。满足“偶方”。

### 主题：匹配关系 - 公众类型对应

**例6**：（2025云南）非组织公众就是在公共关系对象中，处于无组织状态的个体的总和，非组织公众一般包括流散型公众、聚散型公众、周期型公众和固定型公众。
①流散型公众：极为分散的最不稳定的公众；
②聚散型公众：因某一临时事件、活动或某一共同问题临时聚集在一起的公众；
③周期型公众：按一定的规律聚集在一起的公众；
④固定型公众：由于兴趣、爱好、习惯的影响，比较集中地与某些组织、社团、商店、杂志社或某种商品生产厂家保持稳定联系的公众。

根据上述定义，下列对应正确的是：

- A. 乘坐北京开往上海高铁的旅客——④
- B. 定期来某学校上课的函授班学员——③
- C. 经常来社区棋牌室打牌的退休人员——②
- D. 奥运会期间来田径体育场观看比赛的观众——①

<BlurredAnswer>
### 思路剖析

**第一步：分析题型，明确任务。**
这道题是典型的“定义-案例”匹配题。我们需要分析每个选项描述的案例，然后去四个定义中寻找最贴切的一个，最后判断其给出的对应关系是否正确。

**第二步：从案例入手，逐一分析匹配。**

-   **A选项**：“乘坐北京开往上海高铁的旅客”。
    -   **案例分析**：这些旅客因为“乘坐同一趟高铁”这个<span style={{color:'red'}}>临时事件</span>而聚集在一起，到达目的地后便会散开。
    -   **定义匹配**：这完全符合②聚散型公众的“因某一临时事件...临时聚集”的特征。
    -   **判断**：选项将其对应为④固定型公众，错误。

-   **B选项**：“定期来某学校上课的函授班学员”。
    -   **案例分析**：核心词是<span style={{color:'red'}}>“定期”</span>，这体现了一种时间上的规律性。
    -   **定义匹配**：这精准地对应了③周期型公众的“按一定的规律聚集”的特征。
    -   **判断**：选项将其对应为③，正确。

-   **C选项**：“经常来社区棋牌室打牌的退休人员”。
    -   **案例分析**：“经常来”说明了一种<span style={{color:'red'}}>习惯性</span>，“打牌”是一种<span style={{color:'red'}}>兴趣爱好</span>，“社区棋牌室”是一个相对<span style={{color:'red'}}>固定的场所</span>。
    -   **定义匹配**：这完美地符合④固定型公众的“由于兴趣、爱好、习惯的影响...保持稳定联系”的特征。
    -   **判断**：选项将其对应为②聚散型公众，错误。

-   **D选项**：“奥运会期间来田径体育场观看比赛的观众”。

    -   **定义匹配**：这符合②聚散型公众的特征。
    -   **判断**：选项将其对应为①流散型公众（最不稳定，如街头的行人），错误。

### 主题：包含关系 - 层级概念判断

**例7**：（2025国考）主体间性是指人对他人意图的判断与推测。主体间性有不同的级别，一级主体间性即人对另一个人意图的判断与推测，二级主体间性即人对另一人关于其他人意图的判断与推测的认知的认识。

根据上述定义，下列体现了二级主体间性的是：

- A. 小明饿了，去问妈妈饭做好没有，妈妈说爸爸正在做
- B. 老师知道学生们不喜欢做作业，周五就在班级群里通知本周末不留作业
- C. 小王知道老板爱挑毛病，每次提交的总结都得仔细检查好几遍
- D. 小花看见妈妈冲爸爸使了个眼色后，爸爸尴尬地放下酒杯，小花偷偷地笑了

<BlurredAnswer>
### 思路剖析

**第一步：拆解定义，理清层级关系。**
这道题的核心是区分“一级”和“二级”主体间性。
-   **一级主体间性**：结构是“A -> B”。即A推测B的意图。
    -   例子：我（A）觉得你（B）想喝水。
-   **二级主体间性**：结构是“A -> (B -> C)”。即A认识到“B正在推测C的意图”。
    -   这里面至少有三个人（A, B, C），并且有一个认知链条。
    -   例子：我（A）知道你（B）明白他（C）想要什么。

**第二步：分析选项中的人物关系和认知链条。**
题目问的是“二级主体间性”，我们要找一个符合“A -> (B -> C)”结构的选项。

-   **A选项**：小明问妈妈，妈妈告知一个事实（爸爸在做饭）。这里没有体现出任何一方对另一方“意图”的推测。不符合主体间性。

-   **B选项**：老师（A）知道学生们（B）不喜欢做作业（意图）。这是一个典型的“A -> B”结构。属于一级主体间性。

-   **C选项**：小王（A）知道老板（B）爱挑毛病（意图/习惯）。这同样是“A -> B”的结构。属于一级主体间性。

-   **D选项**：我们来分析这个场景中的三个角色：小花（A）、爸爸（B）、妈妈（C）。
    1.  **第一层（B -> C）**：妈妈（C）对爸爸（B）使眼色，爸爸（B）看懂了（推测出意图），于是放下了酒杯。这是一个完整的“一级主体间性”事件，即爸爸推测出了妈妈不让他喝酒的意图。
    2.  **第二层（A -> (B -> C)）**：小花（A）看到了这一切，并且“偷偷地笑了”。这个“笑”表明，小花<span style={{color:'red'}}>认识到了</span>“爸爸已经明白了妈妈的意图”这件事。
    3.  这个结构完美地符合“人（小花）对另一人（爸爸）关于其他人（妈妈）意图的判断与推测的认知的认识”，即“A -> (B -> C)”。

**因此，选择 D。**
</BlurredAnswer>


## 技巧总结

1.  **优先定位，问啥看啥**：题干问哪个定义，就先聚焦于那个定义，暂时忽略其他定义，避免信息干扰。只有在需要对比或者选项非常纠结时，才去看另一个定义。

2.  **拆解要件，逐一比对**：将目标定义的句子进行结构化拆分，提炼出几个关键的构成要素（如：主体、对象、方式、目的、条件、结果等）。然后像过筛子一样，用这些要件去检验每一个选项，但凡有一个要件不满足，就果断排除。

3.  **比较差异，一锤定音**：对于并列关系的多定义，解题的核心就是“找不同”。要快速锁定两个定义之间的<span style={{color:'red'}}>本质区别</span>，这个区别往往就是命题人设置的考点。

4.  **巧用排除，反向思考**：做题时要有排除法的思维。对于“选是题”，要找到那个完全符合所有要件的；对于“选非题”，则要找到那个至少有一个要件不符合的。
