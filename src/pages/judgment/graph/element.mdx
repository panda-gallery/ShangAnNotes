**例：**

### 1.3 元素移动 
> 常常考虑元素的移动，旋转，翻转，平移

**例：**

<Image
  src="https://kaogong-1301372224.cos.ap-nanjing.myqcloud.com/6eb410cb-d734-4e23-b4f2-4e79fb7cb9a2_image.webp"
  alt="element1"
  width={400}
  height={200}

**解析：**
黑白块问题，考虑对称性和数量没有明显差别，考虑元素关系，发现纵向和横线的平移
都不符合，考虑旋转，发现绕图形中点旋转60度，故选择D

### 1.4 元素叠加
> 图形之间比较类似且有明显数量变化，考虑元素叠加  
> 例如：**线段与线段之间叠加相消，黑白叠加变色**

**例：**

<Image
  src="https://kaogong-1301372224.cos.ap-nanjing.myqcloud.com/df06c0bd-541f-421d-ae5a-846ba7883ec2_image.webp"  
  alt="element1"
  width={400}
  height={200}
/>

**解析：**
图形类似，且有明显数量变化，考虑元素叠加，发现相邻图形之间有明显的叠加关系，
发现线段之间叠加相消，故选择B



**例：**

<Image
  src="https://kaogong-1301372224.cos.ap-nanjing.myqcloud.com/830daa4b-22a9-44b2-ad07-815a6dbefc30_image.webp" 
  alt="element1"
  width={400}
  height={200}
/>

**解析：**
图形类似，且有明显数量变化，考虑元素叠加，发现相邻图形之间有明显的叠加关系，
发现为黑白叠加问题：黑 + 白 = 黑，白 + 黑 = 白，黑 + 黑 = 白，白 + 白 = 白，
故选择C

### 1.5 元素分割
> 图形轮廓类似，数量相近

**例：**

<Image
  src="https://kaogong-1301372224.cos.ap-nanjing.myqcloud.com/9d207498-1cba-4845-8953-8db09e5fed46_image.webp"
  alt="element1"
  width={400}
  height={200}
/>

**解析：**
考虑黑白分割，发现图形形状和轮廓，对称一笔画接不满足，考虑黑白叠加，也不满足，图形旋转平移也不满足
考虑分割区域，发现白色区域为3,2,1递减故选择D

## 3. 元素多次变化组合问题
> 图形中元素的种类和个数相近时候，考虑元素的位置，**抓住主要矛盾，观察最大和最小图形的位置进行观察**
> 位置关系以考虑**平移，旋转和翻折**为主

<Image
  src="https://kaogong-1301372224.cos.ap-nanjing.myqcloud.com/b5df240f-eaa0-4fcd-be33-27c7a368d09b_image.webp"
  alt="element1"
  width={400}
  height={200}
/>

**解析：**
可以通过元素的种类和数量排除C，然后判断元素的位置
观察图片中最大的图形（或最小的图形），发现位置关系为先左右对折后上下对折，故答案选择D
