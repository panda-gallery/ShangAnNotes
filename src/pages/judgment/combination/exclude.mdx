# 组合排列：排除法与代入法

在行测判断推理的逻辑判断模块中，组合排列问题因为其信息量大、条件复杂，常常让考生感到头疼。然而，只要掌握了两种核心的解题“利器”——**排除法**和**代入法**，这类看似复杂的问题就能迎刃而解。

## 一、核心概念

### 1. 排除法：当侦探，寻找破绽

#### 核心思想
排除法的本质是“由真推假”。题干给出的所有条件都是确定为真的“线索”，就像侦探手中的证据。我们的任务就是拿着这些板上钉钉的证据，去审视每一个“嫌疑人”（选项），一旦发现某个选项与任何一个证据相冲突，就立刻将其排除。

#### 举个例子
想象一个简单的场景：你要找你的手机。
- **线索1**：你记得出门时手机还在客厅充电。
- **线索2**：你室友说，他看见你的手机不在沙发上。

现在有四个选项：
A. 在卧室的床上
B. 在客厅的充电器上
C. 在客厅的沙发上
D. 在厨房的桌子上

根据这些线索，我们可以这样做：
1.  根据<span style={{color:'red'}}>线索1</span>，手机在“客厅”，所以我们可以直接排除 A (卧室) 和 D (厨房)。
2.  根据<span style={{color:'red'}}>线索2</span>，手机“不在沙发上”，所以我们排除 C。
3.  最后剩下的 B 就是唯一可能的答案。

这就是排除法。它适用于<span style={{color:'red'}}>题干条件确定为真，且选项信息完整</span>的题目。

### 2. 代入法：当实验员，验证假设

#### 核心思想
代入法的本质是“由假验真”。当题干的线索真假难辨，或者线索不足以让我们直接推理时，我们就无法直接排除。这时，我们可以转变思路，把选项当成一个“假设”，然后将这个假设代入到题干的逻辑体系中，看它是否能成立。如果代入后发现与题干的某个条件产生矛盾，那么这个“假设”（选项）就是错误的。

#### 举个例子
还是找东西，但这次情况更复杂：
你面前有甲、乙、丙三个人，他们每个人都说了一句话，但其中只有一个人说的是真话。
- **甲说**：“钥匙在A盒子里。”
- **乙说**：“钥匙不在A盒子里。”
- **丙说**：“钥匙是甲放进去的。”

直接推理很困难，因为我们不知道该信谁。这时就可以用代入法：
1.  **假设A选项（钥匙在A盒子里）是对的**：
    - 如果钥匙在A盒子里，那么甲说的是<span style={{color:'green'}}>真话</span>。
    - 乙说的“不在A盒子里”就是<span style={{color:'red'}}>假话</span>。
    - 这时，已经有了一个真话（甲），那么丙说的也必须是假话。
    - 这个假设（钥匙在A盒子里）能够自圆其说，没有矛盾。

2.  **假设B选项（钥匙不在A盒子里）是对的**：
    - 如果钥匙不在A盒子里，那么甲说的是<span style={{color:'red'}}>假话</span>。
    - 乙说的“不在A盒子里”就是<span style={{color:'green'}}>真话</span>。
    - 同样，丙说的也必须是假话。
    - 这个假设也能自圆其说。

<span style={{color:'orange'}}>（这里我们发现，仅凭前两个条件，两种假设都可能成立，说明还需要丙的条件来进一步判断，这就是代入验证的过程）。</span>

代入法适用于<span style={{color:'red'}}>题干条件有真有假、或包含假设性条件（如果...那么...）、或提问方式为“以下哪项可能为真/假”</span>的题目。

## 二、真题讲解

### 主题1：排除法——多重确定条件的综合应用

**例1**：
赵先生、钱先生、孙先生、李先生四人参加一项技能比赛，获得了比赛的前四名。据了解，他们之间有以下关系：
① 孙先生和李先生经常相约一起打篮球； 
② 第一名和第三名在这次比赛中刚认识； 
③ 第二名不会骑自行车，也不打篮球； 
④ 赵先生的名次比钱先生的名次靠前； 
⑤ 钱先生和李先生每天一起骑自行车上班。

根据以上条件，可以判断此次比赛的第一、二、三、四名次的获得者分别是（）。

- A. 孙先生、赵先生、钱先生、李先生
- B. 李先生、赵先生、孙先生、钱先生
- C. 李先生、孙先生、赵先生、钱先生
- D. 孙先生、李先生、赵先生、钱先生

<BlurredAnswer>
**【思路剖析】**

本题所有条件均为确定信息，适合使用排除法。核心策略是“读一句，排一句”，并优先使用约束性最强的条件。

**第一步：分析条件③和⑤，寻找强约束。**
- 条件③明确指出：“第二名不会骑自行车”。
- 条件⑤指出：“钱先生和李先生每天一起骑自行车上班”。
- <span style={{color:'red'}}>将两个条件结合</span>，可以得出结论：钱先生和李先生都**不可能**是第二名。
- 审查选项：
  - A选项中，第二名是赵先生，符合。
  - B选项中，第二名是赵先生，符合。
  - C选项中，第二名是孙先生，符合。
  - D选项中，第二名是李先生，与我们的结论“李先生不可能是第二名”**矛盾**。因此，<span style={{color:'red'}}>直接排除D选项</span>。

**第二步：分析条件①和③，继续排除。**
- 条件③明确指出：“第二名不打篮球”。
- 条件①指出：“孙先生和李先生经常相约一起打篮球”。
- <span style={{color:'red'}}>将两个条件结合</span>，可以得出结论：孙先生和李先生都**不可能**是第二名。
- 审查剩余选项：
  - C选项中，第二名是孙先生，与我们的结论“孙先生不可能是第二名”**矛盾**。因此，<span style={{color:'red'}}>直接排除C选项</span>。

**第三步：分析条件④，完成排除。**
- 条件④明确指出：“赵先生的名次比钱先生的名次靠前”。
- 审查剩余选项A和B：
  - A选项中，赵先生第二名，钱先生第三名。赵（2）比钱（3）靠前，**符合**。
  - B选项中，赵先生第二名，钱先生第四名。赵（2）比钱（4）靠前，**符合**。
- <span style={{color:'orange'}}>注意：到这一步，条件④没能帮助我们排除选项，说明还有条件没用。</span>

**第四步：回归所有条件，利用条件②。**
- 条件②指出：“第一名和第三名在这次比赛中刚认识”。
- 我们回头看条件①：“孙先生和李先生经常相约一起打篮球”，说明孙先生和李先生是**老相识**。
- 结论：孙先生和李先生**不可能同时**是第一名和第三名。
- 审查剩余选项A和B：
  - A选项中，第一名是孙先生，第三名是钱先生。不确定他们是否认识。
  - B选项中，第一名是李先生，第三名是孙先生。这两人是老相识，与条件②“刚认识”**矛盾**。因此，<span style={{color:'red'}}>直接排除B选项</span>。

**最终结论：**
排除D、C、B后，唯一剩下的A选项即为正确答案。

</BlurredAnswer>

### 主题2：代入法——真假判断类问题的破解之道

**例2**：
三人在一起猜测晚会节目的顺序。
- 甲说：“一班第一个出场，二班第三个出场。”
- 乙说：“三班第一个出场，四班第四个出场。”
- 丙说：“四班第二个出场，一班第三个出场。”

结果公布后，发现他们的预测都<span style={{color:'red'}}>只对了一半</span>。由以上可以推出，节目的正确出场顺序是（）。

- A. 四班第一，三班第二，一班第三，二班第四
- B. 二班第一，一班第二，三班第三，四班第四
- C. 三班第一，四班第二，二班第三，一班第四
- D. 一班第一，二班第二，四班第三，三班第四

<BlurredAnswer>
**【思路剖析】**

本题的<span style={{color:'red'}}>关键信息是“预测都只对了一半”</span>，这意味着每个人的两句话必然是“一真一假”。由于我们无法确定哪句真、哪句假，直接推理非常困难，因此这是使用代入法的典型场景。

**第一步：选择一个选项代入验证。**
我们从A选项开始，假设A选项的顺序“四班第一，三班第二，一班第三，二班第四”是正确的。

**第二步：用假设的正确顺序去验证每个人的话。**
- **验证甲的话**：“一班第一个出场” (错误，正确的是一班第三)，“二班第三个出场” (错误，正确的是二班第四)。
  - 甲的预测是“两假”。这与题干“只对了一半”（一真一假）的条件**矛盾**。
  - 因此，<span style={{color:'red'}}>直接排除A选项</span>。

**第三步：继续代入下一个选项。**
我们代入B选项，假设B选项的顺序“二班第一，一班第二，三班第三，四班第四”是正确的。

**第四步：再次验证每个人的话。**
- **验证甲的话**：“一班第一个出场” (错误，正确的是一班第二)，“二班第三个出场” (错误，正确的是二班第一)。
  - 甲的预测是“两假”。这与题干条件**矛盾**。
  - 因此，<span style={{color:'red'}}>直接排除B选项</span>。

**第五步：继续代入C选项。**
我们代入C选项，假设C选项的顺序“三班第一，四班第二，二班第三，一班第四”是正确的。

**第六步：再次验证每个人的话。**
- **验证甲的话**：“一班第一个出场” (错误)，“二班第三个出场” (正确)。
  - 甲的预测是“一假一真”。<span style={{color:'green'}}>符合题干条件</span>。
- **验证乙的话**：“三班第一个出场” (正确)，“四班第四个出场” (错误)。
  - 乙的预测是“一真一假”。<span style={{color:'green'}}>符合题干条件</span>。
- **验证丙的话**：“四班第二个出场” (正确)，“一班第三个出场” (错误)。
  - 丙的预测是“一真一假”。<span style={{color:'green'}}>符合题干条件</span>。

**最终结论：**
C选项代入后，完美符合题干中“所有人的预测都只对了一半”的要求，因此C为正确答案。

</BlurredAnswer>

### 主题3：综合应用——排除法与代入法的结合

**例3**：
有甲乙丙丁戊五个人参加比赛，比赛结束后，有己庚辛壬癸五个人对他们的名次做了如下判断：
- 己：甲第一名，乙第二名
- 庚：丁第四名，戊第五名
- 辛：丙第三名，乙第二名
- 壬：甲第一名，丁第四名
- 癸：丙第四名，戊第五名

赛后名次公布之后发现，己庚辛每人<span style={{color:'red'}}>最多猜对了一半</span>，壬癸<span style={{color:'red'}}>至少猜对了一半</span>。则甲乙丙丁戊的名次依次是：

- A. 1、3、4、2、5
- B. 2、1、3、4、5
- C. 1、2、4、5、3
- D. 5、3、4、1、2

<BlurredAnswer>
**【思路剖析】**

本题条件复杂，包含多种真假限定，是典型的<span style={{color:'red'}}>代入法</span>应用题。

**第一步：明确代入条件。**
- **己、庚、辛**：每人说的话里，真的数量是0个或1个。
- **壬、癸**：每人说的话里，真的数量是1个或2个。

**第二步：逐一验证选项。**

*   **A选项：甲(1)、乙(3)、丙(4)、丁(2)、戊(5)**
    *   己：甲第一(真)，乙第二(假)。✅ (1真，符合最多对一半)
    *   庚：丁第四(假)，戊第五(真)。✅ (1真，符合最多对一半)
    *   辛：丙第三(假)，乙第二(假)。✅ (0真，符合最多对一半)
    *   壬：甲第一(真)，丁第四(假)。✅ (1真，符合至少对一半)
    *   癸：丙第四(真)，戊第五(真)。✅ (2真，符合至少对一半)
    *   <span style={{color:'green'}}>所有条件均满足，A为正确答案。</span>

*   **B选项：甲(2)、乙(1)、丙(3)、丁(4)、戊(5)**
    *   庚：丁第四(真)，戊第五(真)。❌ (2真，不符合“最多对一半”)
    *   直接排除B。

*   **C选项：甲(1)、乙(2)、丙(4)、丁(5)、戊(3)**
    *   己：甲第一(真)，乙第二(真)。❌ (2真，不符合“最多对一半”)
    *   直接排除C。

*   **D选项：甲(5)、乙(3)、丙(4)、丁(1)、戊(2)**
    *   壬：甲第一(假)，丁第四(假)。❌ (0真，不符合“至少对一半”)
    *   直接排除D。

**最终结论：**
通过代入验证，只有A项完全符合题干的所有真假条件。

</BlurredAnswer>

**例4**：
甲、乙、丙、丁4位同学参加学校运动会。已知他们4人每人都至少获得1个奖项，4人获奖总数为10。关于具体获奖情况，4人还有如下说法：
- 甲：乙和丙获奖总数为5；
- 乙：丙和丁获奖总数为5；
- 丙：丁和甲获奖总数为5；
- 丁：甲和乙获奖总数为4。

后来得知，<span style={{color:'red'}}>获得2个奖项的人说了假话</span>，而其他人均说了真话。根据以上信息，甲、乙、丙、丁4人具体的获奖数分别应是：

- A. 2、3、2、3
- B. 2、4、1、3
- C. 2、2、2、4
- D. 2、2、3、3

<BlurredAnswer>
**【思路剖析】**

本题的真假与条件本身（获奖数）挂钩，是“自指性”的真假话问题，使用<span style={{color:'red'}}>代入法结合排除法</span>是最高效的策略。

**第一步：寻找确定信息。**
观察所有选项，我们发现一个共同点：<span style={{color:'orange'}}>所有选项中甲的获奖数都是2</span>。这是一个可以被当作事实的突破口。

**第二步：根据突破口进行推理。**
- 既然甲的获奖数确定为2，根据规则“获得2个奖项的人说了假话”，我们可以确定：**甲说的是假话**。
- 甲的话是：“乙和丙获奖总数为5”。
- 因为甲说的是假话，所以真实情况是：“<span style={{color:'red'}}>乙和丙获奖总数不等于5</span>”。

**第三步：使用结论排除选项。**
现在我们用“乙+丙 ≠ 5”这个确定条件来筛选选项：
- A选项：乙(3) + 丙(2) = 5。与结论矛盾，**排除**。
- B选项：乙(4) + 丙(1) = 5。与结论矛盾，**排除**。
- C选项：乙(2) + 丙(2) = 4。符合“不等于5”，**保留**。
- D选项：乙(2) + 丙(3) = 5。与结论矛盾，**排除**。

**最终结论：**
仅通过一个突破口，我们就排除了A、B、D三个选项，因此C是唯一可能的正确答案。

</BlurredAnswer>

**例5**：
将6盒茶叶放入甲、乙、丙、丁、戊、己、庚、辛八个箱子中，其中四个箱子有茶叶。已知：
（1）在甲、乙、丙、丁四个箱子中共有5盒茶叶；
（2）在丁、戊、己三个箱子中共有3盒茶叶；
（3）在丙、丁两个箱子中共有2盒茶叶。

根据以上信息，可以得出下列哪项？

- A. 甲箱中至少有1盒
- B. 乙箱中至少有2盒
- C. 己箱中至少有2盒
- D. 戊箱中至少有1盒

<BlurredAnswer>
**【思路剖析】**

本题所有条件均为确定信息，是典型的<span style={{color:'red'}}>排除法（推演法）</span>应用题。我们需要像解数学题一样，通过严谨的逻辑推导，逐步锁定信息。

**第一步：整合宏观信息。**
- 总共有6盒茶叶。
- 条件(1) 甲+乙+丙+丁 = 5 盒。
- 由此可推断：剩下的箱子 (戊、己、庚、辛) 中共有 6 - 5 = 1 盒茶叶。
- 既然 (戊、己、庚、辛) 中总共只有1盒，那么己箱中最多只有1盒。因此<span style={{color:'red'}}>排除C选项</span>（己箱中至少有2盒）。

**第二步：聚焦核心变量。**
- 我们有三个方程：
    1. 甲 + 乙 + 丙 + 丁 = 5
    2. 丁 + 戊 + 己 = 3
    3. 丙 + 丁 = 2
- 并且我们知道 `(戊+己) <= 1` (因为戊己庚辛一共才1盒)。

**第三步：解方程，锁定关键值。**
- 观察条件(2) 和 (3)。
- 从条件(3) `丙+丁=2` 可知，丁最多为2盒。
- 从条件(2) `丁+戊+己=3`，我们知道 `(戊+己) 的和不可能超过1`。
- 如果丁为1，那么 1 + `(戊+己)` = 3，则 `(戊+己)` = 2，与 `(戊+己)<=1` 矛盾。
- 如果丁为0，那么 0 + `(戊+己)` = 3，则 `(戊+己)` = 3，与 `(戊+己)<=1` 矛盾。
- 所以唯一的可能是 <span style={{color:'orange'}}>丁=2盒</span>。
- 当丁=2时，代入(3) `丙+2=2`，得出 <span style={{color:'orange'}}>丙=0盒</span>。
- 当丁=2时，代入(2) `2+戊+己=3`，得出 <span style={{color:'orange'}}>戊+己=1盒</span>。

**第四步：代入求解，得出结论。**
- 我们已经知道 丙=0，丁=2。
- 将它们代入条件(1)：甲 + 乙 + 0 + 2 = 5。
- 化简得：**甲 + 乙 = 3**。
- 题目还说“四个箱子有茶叶”。我们已经确定了丁箱有茶叶，(戊,己)中有一个箱子有茶叶。丙、庚、辛没有茶叶。所以剩下的两个有茶叶的箱子必须是**甲和乙**。
- 既然甲和乙都必须有茶叶（即甲>0, 乙>0），且它们的和是3，那么只有两种可能：
    1. 甲=1，乙=2
    2. 甲=2，乙=1
- 综合这两种情况，我们可以确定的是：<span style={{color:'red'}}>甲箱中至少有1盒，乙箱中也至少有1盒</span>。
- 对比选项：
  - A选项 “甲箱中至少有1盒” 正确。
  - B选项 “乙箱中至少有2盒” 不一定，也可能是1盒。
  - D选项 “戊箱中至少有1盒” 不一定，可能是己有1盒。

**最终结论：**
通过严密的逻辑推导，A选项是唯一可以确定的事实。

## 三、技巧总结

1.  **如何选择方法？**
    *   **优先排除法**：当看到题干所有条件都是确定无误的陈述句时，首先考虑排除法。这是最直接、最高效的方法。
    *   **选择代入法**：当题干出现以下特征时，果断使用代入法：
        *   <span style={{color:'red'}}>条件真假不确定</span>，如“三人中只有一人说真话”、“每人说对一半”。
        *   <span style={{color:'red'}}>提问方式为“可能/不可能”</span>，如“以下哪项可能为真？”。
        *   题干信息量少，而选项信息量大且完整。

2.  **排除法核心技巧**
    *   **读一句，排一句**：不要试图一次性记住所有条件，读完一个条件立刻去核对选项，排除掉不符合的，减轻大脑记忆负担。
    *   **寻找关联，强力突破**：优先处理那些能与其他条件产生强关联的、约束性最强的条件。例如，A和B不能同时出现，或者A和B必须同时出现，这类条件往往能一下排除多个选项。
    *   **善用“逆否命题”**：对于“如果A，那么B”这样的条件，其等价的逆否命题“如果非B，那么非A”同样是强有力的排除工具。

3.  **代入法核心技巧**
    *   **从简入手**：从信息最简单、最明确的选项开始代入，或者从题干中涉及最多的元素所在的选项开始代入，可以提高验证效率。
    *   **标记与核对**：在草稿纸上清晰地写下你代入的假设，然后逐一核对题干条件，标记出“真”或“假”，避免在验证过程中思维混乱。
    *   **找到矛盾，立即停止**：代入法的目的是寻找矛盾。一旦发现当前选项与题干的任何一个条件冲突，就立刻停止对该选项的验证，并将其排除，然后转向下一个选项。

通过掌握这两种方法并勤加练习，考生将能自信地应对逻辑判断中的各类组合排列问题。
