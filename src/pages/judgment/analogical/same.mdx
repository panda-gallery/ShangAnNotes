# 逻辑关系 - 全同关系

## 核心概念

<span style={{color:'red'}}>全同关系，也叫同一关系或等同关系，指的是两个（或多个）词语的含义完全相同，可以相互替换，并且指代的是同一个对象或概念。</span>

就像在生活中，我们说到“土豆”，马上就能想到“马铃薯”，说到“西红柿”，也知道它就是“番茄”。这两个词虽然叫法不同，但指的都是同一种蔬菜。这就是一个典型的全同关系。

我们可以将常见的全同关系细分为以下几种类型：

1.  **同义词关系**：
    *   **概念**：两个词的意义完全相同。
    *   **举例**：
        *   `母亲` 与 `妈妈`
        *   `教师` 与 `老师`
        *   `岳父` 与 `丈人`

2.  **不同称谓关系**：
    *   **概念**：同一个事物，在不同时代、不同地区、不同语言或不同场合有不同的叫法。
    *   **举例**：
        *   **古今称谓**：`南京`（今）与 `金陵`（古）、`私塾`（古）与 `学校`（今）。
        *   **中外/音译称谓**：`麦克风`（音译）与 `话筒`（中）、`维生素`（音译）与 `维他命`（音译）。
        *   **人与名号**：`李白`（名）与 `诗仙`（号）、`刘彻`（名）与 `汉武帝`（谥号）。
        *   **方言与普通话**：`玉米`（普通话）与 `包谷`（西南方言）。

3.  **专有名词与简称/全称**：
    *   **概念**：一个事物的正式全称和它的通用简称。
    *   **举例**：
        *   `北京大学` 与 `北大`
        *   `中华人民共和国` 与 `中国`

4.  **集合与元素**：
    *   **概念**：一个集合的名称与其所有元素的列举是全同关系。
    *   **举例**：
        *   `四季` 与 `春夏秋冬`
        *   `五行` 与 `金木水火土`
        *   `四大发明` 与 `造纸术、印刷术、火药、指南针`

**<span style={{color:'orange'}}>易混淆点辨析</span>**

*   **全同关系 vs. 种属关系**：
    *   **全同关系**：A就是B，B就是A (例如：`番茄`就是`西红柿`)。
    *   **种属关系**：A是B的一种 (例如：`苹果`是`水果`的一种，但`水果`不只是`苹果`)。
*   **全同关系 vs. 并列关系**：
    *   **全同关系**：两个词指向同一个对象 (例如：`春节`和`农历新年`是同一个节日)。
    *   **并列关系**：两个词属于同一类别，但各自独立 (例如：`春节`和`中秋节`都是传统节日，但不是同一个)。

## 真题精讲

### 主题：同义词

**【例1】** (2015年浙江) 岳父：丈人

- A．舅舅：外甥
- B．姨妈：婶婶
- C．伯父：侄子
- D．祖母：奶奶

<BlurredAnswer>
**【解题步骤】**

<span style={{color:'red'}}>第一步：判断题干词语间的逻辑关系。</span>
“岳父”和“丈人”都是指妻子的父亲，二者是同义词，构成全同关系。

<span style={{color:'red'}}>第二步：逐一判断选项词语间的逻辑关系。</span>
- A项：“舅舅”指母亲的兄弟，“外甥”指姐妹的儿子，二者是亲属称谓的对应关系，不是全同关系。排除。
- B项：“姨妈”指母亲的姐妹，“婶婶”指父亲的弟弟的妻子，二者是并列关系（同属长辈女性亲属），但不是全同关系。排除。
- C项：“伯父”指父亲的哥哥，“侄子”指兄弟的儿子，二者是亲属称谓的对应关系，不是全同关系。排除。
- D项：“祖母”和“奶奶”都是指父亲的母亲，二者是同义词，构成全同关系，与题干逻辑关系一致。

**故正确答案为D。**
</BlurredAnswer>

### 主题：古今称谓 + 职业与场所

**【例2】** (2022云南) 私塾：学校：教师

- A. 木箸：筷子：客官
- B. 轿辇：火车：旅客
- C. 客栈：宾馆：掌柜
- D. 医馆：医院：医生

<BlurredAnswer>
**【解题步骤】**

<span style={{color:'red'}}>第一步：判断题干词语间的逻辑关系。</span>
“私塾”是古代的教学场所，“学校”是现代的教学场所，二者在功能上等同，构成全同关系（古今称谓）。“教师”在“学校”里工作，构成职业与工作场所的对应关系。

<span style={{color:'red'}}>第二步：逐一判断选项词语间的逻辑关系。</span>
- A项：“木箸”是用木头做的筷子，“筷子”还包括竹筷、银筷等，“木箸”是“筷子”的一种，二者是种属关系。与题干逻辑关系不一致，排除。
- B项：“轿辇”是古代的人力交通工具，“火车”是现代的机械交通工具，二者功能不同，不是全同关系。排除。
- C项：“客栈”是古代的住宿场所，“宾馆”是现代的住宿场所，二者构成全同关系（古今称谓）。但“掌柜”是古代对店主的称呼，与现代的“宾馆”无法构成职业与场所的对应关系。排除。
- D项：“医馆”是古代的医疗场所，“医院”是现代的医疗场所，二者构成全同关系（古今称谓）。“医生”在“医院”里工作，构成职业与工作场所的对应关系，与题干逻辑关系完全一致。

**故正确答案为D。**
</BlurredAnswer>

### 主题：集合与元素的全同

**【例3】** (2015国考) 春夏秋冬：四季

- A. 喜怒哀乐：情绪
- B. 赤橙黄绿：颜色
- C. 早中晚：一天
- D. 东南西北：四方

<BlurredAnswer>
**【解题步骤】**

<span style={{color:'red'}}>第一步：判断题干词语间的逻辑关系。</span>
“春夏秋冬”是“四季”的全部组成部分，二者在外延上完全等同，构成全同关系。

<span style={{color:'red'}}>第二步：逐一判断选项词语间的逻辑关系。</span>
- A项：“喜怒哀乐”是“情绪”的几种典型表现，但“情绪”还包括“惊”、“恐”等，前者是后者的部分列举，二者是种属关系。排除。
- B项：“赤橙黄绿”是“颜色”的几种，但“颜色”的范围更广，还包括“蓝靛紫”等，二者是种属关系。排除。
- C项：“早中晚”是“一天”的时间划分，但并不完全等同于“一天”的全部，比如还缺少凌晨、深夜等，二者是组成关系，但不是全同。排除。
- D项：“东南西北”是“四方”的全部组成部分，二者在外延上完全等同，构成全同关系，与题干逻辑关系一致。

**故正确答案为D。**
</BlurredAnswer>

**【例4】** (2015春季多省联考) 五行：木火土金水

- A. 五经：诗书礼易大学
- B. 五音：宫商韵征羽
- C. 五色：青黄赤白绿
- D. 五味：酸甜甘辛咸

<BlurredAnswer>
**【解题步骤】**

<span style={{color:'red'}}>第一步：判断题干词语间的逻辑关系。</span>
“五行”就是指“木、火、土、金、水”，二者是集合名称与其全部元素的对应，构成全同关系。

<span style={{color:'red'}}>第二步：逐一判断选项词语间的逻辑关系。</span>
- A项：“五经”指《诗》、《书》、《礼》、《易》、《春秋》，选项中的“大学”属于“四书”之一，不属于“五经”。内容不对应，排除。
- B项：“五音”指“宫、商、角、徵、羽”，选项中的“韵”不属于“五音”。内容不对应，排除。
- C项：中国传统文化中的“五色”指“青、黄、赤、白、黑”，选项中的“绿”不在此列。内容不对应，排除。
- D项：中国传统文化中的“五味”指“酸、甘（甜）、苦、辛、咸”，选项中的“甜”与“甘”意义相同，但缺少了“苦”。<span style={{color:'orange'}}>此题选项存在一定争议，但在四个选项中，D项的对应程度最高，其他三项均有明确的错误。</span>在某些语境下，“五味”也泛指各种味道，D项的列举最为接近。在公考中，选择最优选项。

**故正确答案为D。**
*（注：此题部分考试机构答案有争议，但根据最优原则，D为最可能答案）*
</BlurredAnswer>

### 主题：人与名号

**【例5】** (2021重庆) 刘彻：汉武帝

- A. 李隆基：贞观
- B. 朱棣：明成祖
- C. 赵匡胤：宋太宗
- D. 爱新觉罗·玄烨：康熙

<BlurredAnswer>
**【解题步骤】**

<span style={{color:'red'}}>第一步：判断题干词语间的逻辑关系。</span>
“汉武帝”是“刘彻”的谥号，二者指向同一位皇帝，构成全同关系。

<span style={{color:'red'}}>第二步：逐一判断选项词语间的逻辑关系。</span>
- A项：“贞观”是唐太宗李世民的年号，与唐玄宗“李隆基”无关。排除。
- B项：“明成祖”是“朱棣”的庙号，二者指向同一位皇帝，构成全同关系。保留。
- C项：“宋太宗”是赵光义，“赵匡胤”是宋太祖。二者不是同一个人。排除。
- D项：“康熙”是“爱新觉罗·玄烨”的年号，二者指向同一位皇帝，构成全同关系。保留。

<span style={{color:'red'}}>第三步：对比辨析，寻找更优关系。</span>
B项和D项都构成了全同关系。此时需要进一步细化题干逻辑。题干中的“汉武帝”是<span style={{color:'orange'}}>谥号</span>（皇帝死后评定的称号）。
- B项中的“明成祖”是<span style={{color:'orange'}}>庙号</span>（皇帝死后在太庙中被供奉时所用的名号）。
- D项中的“康熙”是<span style={{color:'orange'}}>年号</span>（皇帝在位时用以纪年的名号）。
谥号和庙号都是死后追封的，而年号是生前使用的。从“死后追封”这一点来看，谥号和庙号的性质比年号更为接近。因此，B项的逻辑关系比D项更贴近题干。

**故正确答案为B。**
</BlurredAnswer>

### 主题：全同关系与矛盾关系

**【例6】** (2024江苏) 停：行：止

- A. 日：月：星
- B. 满：亏：盈
- C. 没：有：无
- D. 萌：荣：枯

<BlurredAnswer>
**【解题步骤】**

<span style={{color:'red'}}>第一步：判断题干词语间的逻辑关系。</span>
“停”和“止”的含义相同，都表示停止，构成<span style={{color:'red'}}>全同关系</span>。“行”指运动、前行。 “停/止”与“行”是两种完全对立的状态，二者构成<span style={{color:'orange'}}>矛盾关系</span>。

<span style={{color:'red'}}>第二步：逐一判断选项词语间的逻辑关系。</span>
- A项：“日”、“月”、“星”是三种不同的天体，三者为并列关系。排除。
- B项：“满”和“盈”是近义词，但“亏”的反义词是“盈”，而不是“满”。逻辑关系不完全一致。排除。
- C项：“没”和“无”的含义相同，都表示不存在，构成<span style={{color:'red'}}>全同关系</span>。“有”表示存在。“没/无”与“有”是两种完全对立的状态，构成<span style={{color:'orange'}}>矛盾关系</span>。与题干逻辑关系完全一致。
- D项：“萌”、“荣”、“枯”描述了植物生长的三个不同阶段，是时间上的顺承关系。排除。

**故正确答案为C。**
</BlurredAnswer>

### 主题：近义词的全同与修饰关系

**【例7】** (2025辽宁) 不根之论：凿空之论：言论

- A. 八斗之才：不羁之才：才华
- B. 得意之作：一家之作：作品
- C. 二姓之好：秦晋之好：和谐
- D. 俯仰之间：喘息之间：时间

<BlurredAnswer>
**【解题步骤】**

<span style={{color:'red'}}>第一步：判断题干词语间的逻辑关系。</span>
“不根之论”指没有根据的言论。“凿空之论”指毫无根据、凭空捏造的言论。二者含义基本相同，构成<span style={{color:'red'}}>全同关系</span>（近义词）。这两个词都是用来修饰“言论”的，与第三个词构成修饰关系。

<span style={{color:'red'}}>第二步：逐一判断选项词语间的逻辑关系。</span>
- A项：“八斗之才”形容才华很高；“不羁之才”形容才华超凡脱俗。二者侧重点不同，不是近义词。排除。
- B项：“得意之作”指自己满意的作品；“一家之作”指自成一派的作品。二者含义不同。排除。
- C项：“二姓之好”和“秦晋之好”都是指两家联姻，构成<span style={{color:'red'}}>全同关系</span>（近义词）。但它们不能用来修饰“和谐”，而是促成“和谐”的方式。与题干逻辑不符。排除。
- D项：“俯仰之间”和“喘息之间”都形容时间极短，构成<span style={{color:'red'}}>全同关系</span>（近义词）。这两个词都是用来修饰“时间”的，与题干逻辑关系完全一致。

**故正确答案为D。**
</BlurredAnswer>

## 技巧总结

1.  **准确认定**：做题第一步，要准确判断题干词语是否为全同关系。核心标准是“A是不是就是B？”。
2.  **细化分类**：如果多个选项都符合全同关系，就要像【例5】一样，进一步细化题干的逻辑关系属于哪一类（如同义词、古今称谓、人与名号等），再从选项中寻找最匹配的一项。
3.  **积累常识**：全同关系题目，特别是涉及古今称谓、名人名号、集合元素的题目，非常考验考生的常识积累。平时要多看多记。
